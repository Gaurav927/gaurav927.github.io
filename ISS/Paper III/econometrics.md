# CHAPTER 1: INTRODUCTION AND OVERVIEW

## 1.1 Position of Econometrics in ISS Examination

The Indian Statistical Service (ISS) Examination is conducted by UPSC for recruitment to the Indian Statistical Service. Econometrics forms a crucial component of the examination, primarily appearing in Paper IV (Statistics III).

### Examination Structure

| Paper | Subject | Total Marks | Econometrics Component |
|-------|---------|-------------|------------------------|
| Paper I | General English | 100 | Nil |
| Paper II | Statistics I | 200 | Nil |
| Paper III | Statistics II | 200 | Nil |
| Paper IV | Statistics III | 200 | 50-70 marks (25-35%) |

### 1.2 Importance of Econometrics

Econometrics carries significant weightage in Paper IV and often determines the difference between selection and non-selection. The subject requires:

- Strong mathematical foundation
- Understanding of statistical inference
- Matrix algebra proficiency
- Ability to connect theory with applications

### 1.3 Nature of Questions

Questions in econometrics section are typically:

1. **Theoretical**: Requiring conceptual understanding and explanations
2. **Derivational**: Requiring mathematical proofs and derivations
3. **Numerical**: Requiring computational skills and application
4. **Analytical**: Requiring comparison and critical analysis

## 5.2 Type 2: Derivation Questions

### Characteristics
- Mathematical proofs required
- Step-by-step derivation expected
- Tests mathematical rigor
- Usually carries 15-20 marks

### Examples

**Example 1:** (20 marks)
"Derive the OLS estimator in matrix form. Prove that it is BLUE under Gauss-Markov assumptions."

**Expected Answer Structure:**

**Part A: Derivation of OLS Estimator**

Given the model: Y = Xβ + u

Where:
- Y is n×1 vector of observations
- X is n×k matrix of regressors
- β is k×1 vector of parameters
- u is n×1 vector of disturbances

Minimize: S = u'u = (Y - Xβ)'(Y - Xβ)

Expanding:
S = Y'Y - 2β'X'Y + β'X'Xβ

Taking derivative with respect to β:
∂S/∂β = -2X'Y + 2X'Xβ = 0

Solving:
X'Xβ = X'Y
β̂ = (X'X)⁻¹X'Y

**Part B: Proof of BLUE Property**

**Unbiasedness:**
E(β̂) = E[(X'X)⁻¹X'Y]
     = E[(X'X)⁻¹X'(Xβ + u)]
     = β + (X'X)⁻¹X'E(u)
     = β (since E(u) = 0)

**Best (Minimum Variance):**
Var(β̂) = E[(β̂ - β)(β̂ - β)']
       = E[(X'X)⁻¹X'uu'X(X'X)⁻¹]
       = (X'X)⁻¹X'E(uu')X(X'X)⁻¹
       = σ²(X'X)⁻¹

For any other linear unbiased estimator β̃ = CY:
Var(β̃) - Var(β̂) is positive semi-definite

**Example 2:** (18 marks)
"Derive the 2SLS estimator. Show that it is consistent."

**Expected Answer Structure:**

**Stage 1:**
Regress endogenous RHS variables on all exogenous variables:
Ŷ₂ = X(X'X)⁻¹X'Y₂ = PₓY₂

**Stage 2:**
Replace Y₂ with Ŷ₂ in structural equation:
Y₁ = Ŷ₂β + X₁γ + u

Let Z = [Y₂, X₁] and Ẑ = [Ŷ₂, X₁]

β̂₂ₛₗₛ = (Ẑ'Z)⁻¹Ẑ'Y₁

**Consistency Proof:**
Show that plim(Ẑ'u/n) = 0 and plim(Ẑ'Z/n) exists and is non-singular.

**Example 3:** (15 marks)
"Derive the Koyck transformation from an infinite distributed lag model."

**Expected Answer Structure:**

Starting model:
Yₜ = α + β₀Xₜ + β₁Xₜ₋₁ + β₂Xₜ₋₂ + ... + uₜ

Assume geometric decay: βᵢ = β₀λⁱ where 0 < λ < 1

Yₜ = α + β₀(Xₜ + λXₜ₋₁ + λ²Xₜ₋₂ + ...) + uₜ ... (1)

Lag equation (1) by one period and multiply by λ:
λYₜ₋₁ = λα + β₀(λXₜ₋₁ + λ²Xₜ₋₂ + ...) + λuₜ₋₁ ... (2)

Subtract (2) from (1):
Yₜ - λYₜ₋₁ = α(1-λ) + β₀Xₜ + uₜ - λuₜ₋₁

**Koyck Model:**
Yₜ = α(1-λ) + β₀Xₜ + λYₜ₋₁ + vₜ

Where vₜ = uₜ - λuₜ₋₁ (MA(1) error)

## 5.3 Type 3: Numerical/Application Questions

### Characteristics
- Data-based calculations
- Application of theoretical concepts
- Tests computational skills
- Usually carries 15-25 marks

### Examples

**Example 1:** (20 marks)
"Consider the following simultaneous equation model:

Demand: Q = α₀ + α₁P + α₂Y + u₁
Supply: Q = β₀ + β₁P + β₂W + u₂

Where Q = quantity, P = price, Y = income, W = weather

(a) Write the model in matrix form
(b) Identify the reduced form equations
(c) Check identification of each equation using order and rank conditions
(d) Suggest appropriate estimation method for each equation"

**Expected Answer Structure:**

**(a) Matrix Form:**

Structural form: BY + ΓX = U

Where:
```
B = [1   -α₁]    Y = [Q]    Γ = [-α₀  -α₂   0 ]    X = [1]    U = [u₁]
    [1   -β₁]        [P]        [-β₀   0   -β₂]        [Y]        [u₂]
                                                        [W]
```

**(b) Reduced Form:**

Y = -B⁻¹ΓX = ΠX

Solving:
```
Q = π₁₀ + π₁₁Y + π₁₂W
P = π₂₀ + π₂₁Y + π₂₂W
```

**(c) Identification:**

**Demand Equation:**
- Endogenous variables (m) = 2 (Q, P)
- Exogenous variables in equation (k) = 1 (Y)
- Total exogenous variables (K) = 2 (Y, W)

Order condition: K - k ≥ m - 1
2 - 1 ≥ 2 - 1
1 ≥ 1 ✓ (Exactly identified)

Rank condition: Check rank of matrix of excluded variables' coefficients in other equations.
Excluded variable: W
Coefficient in supply equation: -β₂ ≠ 0
Rank = 1 = m - 1 ✓

**Supply Equation:**
- Excluded variable: Y
- Coefficient in demand equation: -α₂ ≠ 0
- Order: 1 ≥ 1 ✓
- Rank = 1 ✓ (Exactly identified)

**(d) Estimation Method:**
Both equations are exactly identified.
- Can use ILS (Indirect Least Squares)
- Can use 2SLS
- 2SLS is preferred for robustness

**Example 2:** (15 marks)
"Given the following data for a regression model Y = β₀ + β₁X + u:

| Observation | Y | X |
|-------------|---|---|
| 1 | 10 | 2 |
| 2 | 15 | 3 |
| 3 | 18 | 4 |
| 4 | 22 | 5 |
| 5 | 28 | 6 |

(a) Estimate β₀ and β₁ using OLS
(b) Calculate residuals
(c) Compute Durbin-Watson statistic
(d) Test for autocorrelation at 5% level"

**Expected Answer Structure:**

**(a) OLS Estimation:**

Calculations:
- n = 5
- ΣX = 20, ΣY = 93
- X̄ = 4, Ȳ = 18.6
- ΣXY = 412
- ΣX² = 90

β̂₁ = [ΣXY - nX̄Ȳ] / [ΣX² - nX̄²]
   = [412 - 5(4)(18.6)] / [90 - 5(16)]
   = [412 - 372] / [90 - 80]
   = 40/10 = 4

β̂₀ = Ȳ - β̂₁X̄ = 18.6 - 4(4) = 2.6

**Estimated Model: Ŷ = 2.6 + 4X**

**(b) Residuals:**

| Obs | Y | Ŷ | e = Y - Ŷ |
|-----|---|---|-----------|
| 1 | 10 | 10.6 | -0.6 |
| 2 | 15 | 14.6 | 0.4 |
| 3 | 18 | 18.6 | -0.6 |
| 4 | 22 | 22.6 | -0.6 |
| 5 | 28 | 26.6 | 1.4 |

**(c) Durbin-Watson Statistic:**

d = Σ(eₜ - eₜ₋₁)² / Σeₜ²

| t | eₜ | eₜ - eₜ₋₁ | (eₜ - eₜ₋₁)² | eₜ² |
|---|-----|-----------|--------------|-----|
| 1 | -0.6 | - | - | 0.36 |
| 2 | 0.4 | 1.0 | 1.00 | 0.16 |
| 3 | -0.6 | -1.0 | 1.00 | 0.36 |
| 4 | -0.6 | 0.0 | 0.00 | 0.36 |
| 5 | 1.4 | 2.0 | 4.00 | 1.96 |

Σ(eₜ - eₜ₋₁)² = 6.00
Σeₜ² = 3.20

d = 6.00/3.20 = 1.875

**(d) Test for Autocorrelation:**

For n = 5, k = 1 (excluding intercept), α = 0.05:
- dL ≈ 0.61
- dU ≈ 1.40

Decision Rules:
- If d < dL: Positive autocorrelation
- If d > dU: No positive autocorrelation
- If dL < d < dU: Inconclusive

Since d = 1.875 > dU = 1.40, we conclude no significant positive autocorrelation.

# CHAPTER 6: MOST FREQUENTLY ASKED QUESTIONS

## 6.1 Guaranteed Questions (100% Frequency)

### Question Set 1: Identification in SEM

**Q1.1:** "Explain the identification problem in simultaneous equations model."

**Model Answer:**

**Introduction:**
The identification problem arises when we try to estimate structural parameters from reduced form parameters. It concerns whether unique values of structural parameters can be obtained from the estimated reduced form coefficients.

**Structural vs Reduced Form:**

Consider a two-equation model:
```
Structural Form:
Y₁ = β₁₂Y₂ + γ₁₁X₁ + u₁
Y₂ = β₂₁Y₁ + γ₂₂X₂ + u₂

Reduced Form:
Y₁ = π₁₁X₁ + π₁₂X₂ + v₁
Y₂ = π₂₁X₁ + π₂₂X₂ + v₂
```

**The Problem:**
- We can estimate π's consistently using OLS on reduced form
- But we need to recover β's and γ's from π's
- This may not always be possible uniquely

**Types of Identification:**

1. **Under-identified:** Cannot obtain unique structural parameters
   - Infinite solutions exist
   - Equation cannot be estimated

2. **Exactly (Just) Identified:** Unique structural parameters obtainable
   - One solution exists
   - Can use ILS or 2SLS

3. **Over-identified:** More than enough information available
   - Multiple ways to calculate parameters
   - Use 2SLS, which provides optimal combination

**Why OLS Fails:**
In structural equations, endogenous RHS variables are correlated with the error term:
```
E(Y₂u₁) ≠ 0
```
This violates the OLS assumption, making OLS inconsistent.

**Q1.2:** "State and prove the rank and order conditions for identification."

**Model Answer:**

**General Setup:**

Consider a system of G equations with G endogenous variables and K exogenous variables.

For the ith equation:
```
y₁ᵢβᵢ + Y₁ᵢΓᵢ + X₁ᵢδᵢ = uᵢ
```

Where:
- y₁ᵢ = dependent endogenous variable
- Y₁ᵢ = other endogenous variables in equation (gᵢ - 1 variables)
- X₁ᵢ = exogenous variables in equation (kᵢ variables)

**ORDER CONDITION (Necessary but not Sufficient):**

**Statement:**
For the ith equation to be identified:
```
K - kᵢ ≥ gᵢ - 1
```

Or equivalently:
```
Number of excluded exogenous variables ≥ Number of included endogenous variables - 1
```

**Interpretation:**
- K - kᵢ = number of exogenous variables excluded from equation i
- gᵢ - 1 = number of endogenous variables on RHS of equation i

**Classification:**
- If K - kᵢ < gᵢ - 1: Under-identified
- If K - kᵢ = gᵢ - 1: Exactly identified
- If K - kᵢ > gᵢ - 1: Over-identified

**RANK CONDITION (Necessary and Sufficient):**

**Statement:**
The ith equation is identified if and only if:
```
rank(Aᵢ) = G - 1
```

Where Aᵢ is the matrix formed by the coefficients of variables (both endogenous and exogenous) excluded from the ith equation but included in other equations.

**Construction of Aᵢ:**

Step 1: Write the complete system in matrix form: AY + BX = U

Step 2: For equation i, identify excluded variables

Step 3: Form matrix Aᵢ using coefficients of excluded variables from other G-1 equations

Step 4: Check if rank(Aᵢ) = G - 1

**Proof of Necessity:**

For identification, we need to uniquely solve for structural parameters from reduced form. This requires the system of equations relating structural to reduced form parameters to have a unique solution. This is possible only when the rank condition is satisfied.

**Example:**

Consider:
```
Y₁ = β₁₂Y₂ + γ₁₁X₁ + u₁  ... (1)
Y₂ = β₂₁Y₁ + γ₂₂X₂ + u₂  ... (2)
```

**For Equation (1):**
- G = 2, K = 2
- gᵢ = 2 (Y₁, Y₂), kᵢ = 1 (X₁)
- Excluded exogenous: X₂

Order Condition: K - kᵢ ≥ gᵢ - 1
2 - 1 ≥ 2 - 1
1 ≥ 1 ✓ (Exactly identified)

Rank Condition:
Matrix of excluded variable coefficients in other equations:
Aᵢ = [γ₂₂] (coefficient of X₂ in equation 2)
rank(Aᵢ) = 1 = G - 1 ✓

**Equation (1) is exactly identified.**

### Question Set 2: 2SLS Estimation

**Q2.1:** "Derive the Two-Stage Least Squares (2SLS) estimator and prove its consistency."

**Model Answer:**

**Setup:**

Consider a structural equation:
```
y₁ = Y₁β + X₁γ + u
   = Zδ + u
```

Where:
- y₁ = n×1 vector of dependent variable observations
- Y₁ = n×g matrix of endogenous RHS variables
- X₁ = n×k matrix of exogenous variables in equation
- Z = [Y₁, X₁] = n×(g+k) matrix of all RHS variables
- δ = [β', γ']' = parameter vector
- u = n×1 error vector

**Problem with OLS:**
```
E(Y₁'u) ≠ 0
```
OLS estimator δ̂ₒₗₛ = (Z'Z)⁻¹Z'y₁ is inconsistent.

**2SLS DERIVATION:**

**Stage 1: Purging Endogenous Variables**

Regress each column of Y₁ on all exogenous variables X in the system:
```
Ŷ₁ = X(X'X)⁻¹X'Y₁ = PₓY₁
```

Where:
- X = n×K matrix of all exogenous variables in system
- Pₓ = X(X'X)⁻¹X' is the projection matrix

The fitted values Ŷ₁ are:
- Linear combinations of exogenous variables only
- Uncorrelated with u (asymptotically)

**Stage 2: Estimation Using Purged Variables**

Replace Y₁ with Ŷ₁ in the structural equation:
```
y₁ = Ŷ₁β + X₁γ + u*
```

Define:
```
Ẑ = [Ŷ₁, X₁]
```

Note: X₁ remains unchanged since X₁ = PₓX₁ (exogenous variables project onto themselves)

**2SLS Estimator:**
```
δ̂₂ₛₗₛ = (Ẑ'Z)⁻¹Ẑ'y₁
```

**Alternative Expression:**
```
δ̂₂ₛₗₛ = (Z'PₓZ)⁻¹Z'Pₓy₁
```

**PROOF OF CONSISTENCY:**

**Step 1:** Express the estimator in terms of true parameters
```
δ̂₂ₛₗₛ = (Ẑ'Z)⁻¹Ẑ'y₁
       = (Ẑ'Z)⁻¹Ẑ'(Zδ + u)
       = δ + (Ẑ'Z)⁻¹Ẑ'u
```

**Step 2:** Take probability limits
```
plim(δ̂₂ₛₗₛ) = δ + plim(Ẑ'Z/n)⁻¹ · plim(Ẑ'u/n)
```

**Step 3:** Show plim(Ẑ'u/n) = 0

Since Ẑ = [Ŷ₁, X₁] and:
- Ŷ₁ = PₓY₁ is a linear combination of X
- X₁ is exogenous

We have:
```
plim(Ŷ₁'u/n) = plim(Y₁'Pₓu/n) = 0
plim(X₁'u/n) = 0
```

Therefore: plim(Ẑ'u/n) = 0

**Step 4:** Show plim(Ẑ'Z/n) exists and is non-singular

Under standard regularity conditions:
```
plim(Ẑ'Z/n) = Qẑz (finite, positive definite matrix)
```

**Conclusion:**
```
plim(δ̂₂ₛₗₛ) = δ + Qẑz⁻¹ · 0 = δ
```

**The 2SLS estimator is consistent.**

**Variance of 2SLS Estimator:**
```
Var(δ̂₂ₛₗₛ) = σ²(Ẑ'Z)⁻¹Ẑ'ZZ'Ẑ(Ẑ'Z)⁻¹
           = σ²(Z'PₓZ)⁻¹
```

Estimated as:
```
V̂ar(δ̂₂ₛₗₛ) = σ̂²(Ẑ'Z)⁻¹
```

Where σ̂² = (y₁ - Zδ̂₂ₛₗₛ)'(y₁ - Zδ̂₂ₛₗₛ)/(n - g - k)

### Question Set 3: OLS and Gauss-Markov Theorem

**Q3.1:** "State and prove the Gauss-Markov theorem."

**Model Answer:**

**Statement of Gauss-Markov Theorem:**

Under the assumptions of the Classical Linear Regression Model, the Ordinary Least Squares (OLS) estimator is the Best Linear Unbiased Estimator (BLUE) of the regression coefficients.

**Assumptions (Gauss-Markov Conditions):**

1. **Linearity:** Y = Xβ + u
2. **Full Rank:** X is n×k matrix with rank k (full column rank)
3. **Exogeneity:** E(u|X) = 0, implying E(u) = 0
4. **Spherical Disturbances:** Var(u|X) = σ²Iₙ
   - Homoscedasticity: Var(uᵢ) = σ² for all i
   - No autocorrelation: Cov(uᵢ, uⱼ) = 0 for i ≠ j

**Note:** Normality is NOT required for Gauss-Markov theorem.

**OLS Estimator:**
```
β̂ = (X'X)⁻¹X'Y
```

**PROOF:**

**Part 1: Linearity**

β̂ = (X'X)⁻¹X'Y = CY

Where C = (X'X)⁻¹X' is a k×n matrix of constants (given X).

Therefore, β̂ is a linear function of Y. ✓

**Part 2: Unbiasedness**

```
β̂ = (X'X)⁻¹X'Y
   = (X'X)⁻¹X'(Xβ + u)
   = (X'X)⁻¹X'Xβ + (X'X)⁻¹X'u
   = β + (X'X)⁻¹X'u
```

Taking expectation:
```
E(β̂) = β + (X'X)⁻¹X'E(u)
     = β + (X'X)⁻¹X' · 0
     = β
```

Therefore, β̂ is unbiased. ✓

**Part 3: Best (Minimum Variance)**

**Variance of OLS:**
```
Var(β̂) = E[(β̂ - β)(β̂ - β)']
       = E[(X'X)⁻¹X'u · u'X(X'X)⁻¹]
       = (X'X)⁻¹X' E(uu') X(X'X)⁻¹
       = (X'X)⁻¹X' (σ²I) X(X'X)⁻¹
       = σ²(X'X)⁻¹X'X(X'X)⁻¹
       = σ²(X'X)⁻¹
```

**Consider any other linear unbiased estimator:**

Let β̃ = AY be another linear unbiased estimator.

For unbiasedness:
```
E(β̃) = E(AY) = E(A(Xβ + u)) = AXβ + AE(u) = AXβ = β
```

This requires AX = Iₖ.

Write A = (X'X)⁻¹X' + D, where D is some k×n matrix.

For AX = Iₖ:
```
[(X'X)⁻¹X' + D]X = Iₖ
(X'X)⁻¹X'X + DX = Iₖ
Iₖ + DX = Iₖ
DX = 0
```

**Variance of β̃:**
```
Var(β̃) = Var(AY) = A · Var(Y) · A'
       = A(σ²I)A' = σ²AA'
       = σ²[(X'X)⁻¹X' + D][(X'X)⁻¹X' + D]'
       = σ²[(X'X)⁻¹X' + D][X(X'X)⁻¹ + D']
       = σ²[(X'X)⁻¹X'X(X'X)⁻¹ + (X'X)⁻¹X'D' + DX(X'X)⁻¹ + DD']
       = σ²[(X'X)⁻¹ + 0 + 0 + DD']    (since DX = 0)
       = σ²(X'X)⁻¹ + σ²DD'
       = Var(β̂) + σ²DD'
```

Since DD' is positive semi-definite:
```
Var(β̃) - Var(β̂) = σ²DD' ≥ 0
```

This means Var(β̃) ≥ Var(β̂) for any linear unbiased estimator β̃.

**Therefore, β̂ₒₗₛ has minimum variance among all linear unbiased estimators.**

**Conclusion:** β̂ₒₗₛ = (X'X)⁻¹X'Y is BLUE. ∎

## 6.2 High Frequency Questions (85-95% Frequency)

### Question Set 4: Heteroscedasticity

**Q4.1:** "What are the consequences of heteroscedasticity on OLS estimators? Describe methods to detect heteroscedasticity."

**Model Answer:**

**Definition:**
Heteroscedasticity occurs when the variance of the error term is not constant across observations:
```
Var(uᵢ) = σᵢ² ≠ σ² for all i
```

**CONSEQUENCES ON OLS ESTIMATORS:**

| Property | Effect of Heteroscedasticity |
|----------|------------------------------|
| Unbiasedness | OLS remains unbiased |
| Consistency | OLS remains consistent |
| Efficiency | OLS is no longer efficient (not BLUE) |
| Variance Estimation | σ̂² is biased |
| Standard Errors | Incorrect (usually underestimated) |
| t-statistics | Invalid |
| F-statistics | Invalid |
| Confidence Intervals | Invalid |
| Hypothesis Tests | Misleading conclusions |

**Detailed Explanation:**

1. **OLS Still Unbiased:**
   ```
   E(β̂) = E[(X'X)⁻¹X'Y] = β
   ```
   Unbiasedness doesn't depend on homoscedasticity.

2. **OLS No Longer Efficient:**
   
   True variance under heteroscedasticity:
   ```
   Var(β̂) = (X'X)⁻¹X'ΩX(X'X)⁻¹
   ```
   Where Ω = diag(σ₁², σ₂², ..., σₙ²)
   
   This is NOT equal to σ²(X'X)⁻¹

3. **Variance Estimator Biased:**
   
   The usual estimator s² = Σeᵢ²/(n-k) is biased for any single σᵢ².

**DETECTION METHODS:**

**A. Graphical Methods:**

1. **Residual Plot:**
   - Plot êᵢ against Ŷᵢ or Xᵢ
   - Look for fan-shaped or systematic patterns
   - Random scatter suggests homoscedasticity

2. **Squared Residual Plot:**
   - Plot êᵢ² against Ŷᵢ or Xᵢ
   - Increasing/decreasing pattern indicates heteroscedasticity

**B. Formal Tests:**

1. **Goldfeld-Quandt Test:**
   
   Procedure:
   - Order observations by suspected variable
   - Omit middle c observations
   - Run separate regressions on first and last (n-c)/2 observations
   - Compute F = RSS₂/RSS₁ (larger variance in numerator)
   - Compare with F[(n-c-2k)/2, (n-c-2k)/2]
   
   H₀: Homoscedasticity
   H₁: Heteroscedasticity

2. **Breusch-Pagan Test:**
   
   Procedure:
   - Estimate model and obtain residuals êᵢ
   - Compute σ̂² = Σêᵢ²/n
   - Regress êᵢ²/σ̂² on suspected variables Z
   - Test statistic: LM = (1/2) × ESS from auxiliary regression
   - Under H₀: LM ~ χ²(p) where p = number of variables in Z

3. **White's Test:**
   
   Procedure:
   - Estimate model and obtain residuals êᵢ
   - Regress êᵢ² on all regressors, their squares, and cross-products
   - Test statistic: nR² from auxiliary regression
   - Under H₀: nR² ~ χ²(q) where q = number of regressors in auxiliary regression
   
   Advantage: Does not require specification of heteroscedasticity form

4. **Park Test:**
   
   Model: ln(σᵢ²) = α + β ln(Xᵢ) + vᵢ
   
   Procedure:
   - Estimate model and obtain residuals êᵢ
   - Regress ln(êᵢ²) on ln(Xᵢ)
   - Test significance of β

5. **Glejser Test:**
   
   Procedure:
   - Estimate model and obtain residuals êᵢ
   - Regress |êᵢ| on various functions of Xᵢ:
     - |êᵢ| = α + βXᵢ + vᵢ
     - |êᵢ| = α + β√Xᵢ + vᵢ
     - |êᵢ| = α + β/Xᵢ + vᵢ
   - Test significance of β

### Question Set 5: Autocorrelation

**Q5.1:** "Discuss the consequences of autocorrelation on OLS estimators. Derive the Durbin-Watson test statistic."

**Model Answer:**

**Definition:**
Autocorrelation (serial correlation) occurs when error terms are correlated across observations:
```
Cov(uᵢ, uⱼ) ≠ 0 for i ≠ j
```

Most common form - AR(1):
```
uₜ = ρuₜ₋₁ + εₜ, |ρ| < 1
```
Where εₜ ~ iid(0, σ²ε)

**CONSEQUENCES ON OLS ESTIMATORS:**

| Property | Effect of Autocorrelation |
|----------|---------------------------|
| Unbiasedness | OLS remains unbiased |
| Consistency | OLS remains consistent |
| Efficiency | OLS is no longer efficient (not BLUE) |
| Variance Estimation | Usually underestimated |
| Standard Errors | Too small (for positive autocorrelation) |
| t-statistics | Inflated |
| F-statistics | Inflated |
| R² | Inflated |
| Confidence Intervals | Too narrow |
| Type I Error | Increased (reject H₀ too often) |

**Detailed Explanation:**

1. **True Variance Under AR(1):**
   
   For simple regression Yₜ = β₀ + β₁Xₜ + uₜ:
   ```
   Var(β̂₁) = σ²ᵤ/Σ(Xₜ - X̄)² × [1 + 2ρ(r₁) + 2ρ²(r₂) + ...]
   ```
   Where rⱼ = Σ(Xₜ - X̄)(Xₜ₋ⱼ - X̄)/Σ(Xₜ - X̄)²
   
   For positive ρ and positive rⱼ, the true variance exceeds the OLS-estimated variance.

2. **Why Variance is Underestimated:**
   
   OLS assumes:
   ```
   Var(β̂) = σ²(X'X)⁻¹
   ```
   
   But true variance is:
   ```
   Var(β̂) = σ²(X'X)⁻¹X'ΩX(X'X)⁻¹
   ```
   
   Where Ω is the autocorrelation matrix with ρ|i-j| as (i,j)th element.

3. **Inflated R²:**
   
   With positive autocorrelation, successive residuals tend to have the same sign, making the model appear to fit better than it actually does.

**DURBIN-WATSON TEST:**

**Assumptions:**
- Model includes an intercept
- Regressors are non-stochastic
- No lagged dependent variable
- Testing for AR(1) autocorrelation

**Derivation of Test Statistic:**

The Durbin-Watson statistic is defined as:

```
d = Σₜ₌₂ⁿ (eₜ - eₜ₋₁)² / Σₜ₌₁ⁿ eₜ²
```

Where eₜ are OLS residuals.

**Expanding the numerator:**

```
Σ(eₜ - eₜ₋₁)² = Σeₜ² - 2Σeₜeₜ₋₁ + Σeₜ₋₁²
```

For large n:
```
Σeₜ² ≈ Σeₜ₋₁²
```

Therefore:
```
d ≈ (2Σeₜ² - 2Σeₜeₜ₋₁) / Σeₜ²
  = 2[1 - Σeₜeₜ₋₁/Σeₜ²]
  = 2[1 - ρ̂]
```

Where ρ̂ = Σeₜeₜ₋₁/Σeₜ² is the estimated first-order autocorrelation coefficient.

**Relationship with ρ:**
```
d ≈ 2(1 - ρ̂)
```

**Range of d:**
- When ρ = 0: d ≈ 2 (no autocorrelation)
- When ρ = +1: d ≈ 0 (perfect positive autocorrelation)
- When ρ = -1: d ≈ 4 (perfect negative autocorrelation)

Therefore: 0 ≤ d ≤ 4

**Decision Rules:**

```
                    Positive        Inconclusive    No              Inconclusive    Negative
                    Autocorrelation                 Autocorrelation                 Autocorrelation
    |_______________|_______________|_______________|_______________|_______________|
    0               dL              dU              2               4-dU            4-dL            4

    Reject H₀       Zone of         Accept H₀       Zone of         Reject H₀
    (ρ > 0)         Indecision      (ρ = 0)         Indecision      (ρ < 0)
```

**Critical Values:**
- dL = lower critical value
- dU = upper critical value
- Values depend on n, k, and significance level α

**Hypotheses:**

For positive autocorrelation:
- H₀: ρ = 0
- H₁: ρ > 0

Decision:
- If d < dL: Reject H₀ (positive autocorrelation exists)
- If d > dU: Do not reject H₀
- If dL ≤ d ≤ dU: Inconclusive

**Limitations of Durbin-Watson Test:**

1. Inconclusive region may be large for small samples
2. Not valid when lagged dependent variable is present
3. Only tests for AR(1) autocorrelation
4. Requires non-stochastic regressors
5. Critical values depend on X matrix (bounds test)
6. Assumes normally distributed errors
7. Model must include an intercept

**Alternative Tests:**

1. **Durbin's h-test:** For models with lagged dependent variable
2. **Breusch-Godfrey LM test:** For higher-order autocorrelation
3. **Ljung-Box Q-test:** For multiple autocorrelation lags

### Question Set 6: Multicollinearity

**Q6.1:** "What is multicollinearity? Discuss its consequences and methods to detect it."

**Model Answer:**

**Definition:**

Multicollinearity refers to the existence of linear relationships among explanatory variables in a regression model.

**Types:**

1. **Perfect Multicollinearity:**
   - Exact linear relationship exists: c₁X₁ + c₂X₂ + ... + cₖXₖ = 0
   - (X'X) is singular
   - OLS cannot be computed
   - Rarely occurs in practice

2. **Imperfect (Near) Multicollinearity:**
   - High but not perfect correlation among regressors
   - (X'X) is near-singular (ill-conditioned)
   - OLS can be computed but is unreliable
   - Common in practice

**CONSEQUENCES OF MULTICOLLINEARITY:**

| Aspect | Effect |
|--------|--------|
| OLS Estimators | Still BLUE |
| Unbiasedness | Retained |
| Consistency | Retained |
| Variances of Estimates | Very large |
| Standard Errors | Very large |
| Confidence Intervals | Very wide |
| t-statistics | Small (insignificant) |
| Individual Significance | Coefficients appear insignificant |
| Joint Significance | F-test may be significant |
| R² | May be high |
| Sensitivity | Estimates sensitive to small changes in data |
| Sign of Coefficients | May be wrong |
| Prediction | May still be good if multicollinearity persists |

**The Multicollinearity Paradox:**
- High R² (model fits well overall)
- Significant F-statistic (variables jointly significant)
- But few or no significant t-statistics (variables individually insignificant)

**Mathematical Explanation:**

For the model Y = Xβ + u, the variance of β̂ⱼ is:

```
Var(β̂ⱼ) = σ² / [(1 - Rⱼ²) × Σ(Xⱼᵢ - X̄ⱼ)²]
         = σ² × VIFⱼ / Σ(Xⱼᵢ - X̄ⱼ)²
```

Where:
- Rⱼ² = R² from regressing Xⱼ on all other X variables
- VIFⱼ = 1/(1 - Rⱼ²) = Variance Inflation Factor

As Rⱼ² → 1 (high multicollinearity), VIFⱼ → ∞, and Var(β̂ⱼ) → ∞

**DETECTION METHODS:**

**1. Examination of Correlation Matrix:**

- Compute pairwise correlations between all X variables
- High correlations (|r| > 0.8 or 0.9) suggest multicollinearity
- Limitation: May miss multicollinearity involving more than two variables

**2. Variance Inflation Factor (VIF):**

```
VIFⱼ = 1 / (1 - Rⱼ²)
```

Rules of thumb:
- VIF = 1: No multicollinearity
- VIF > 5: Moderate multicollinearity
- VIF > 10: Severe multicollinearity

**3. Tolerance:**

```
TOLⱼ = 1 - Rⱼ² = 1/VIFⱼ
```

- TOL < 0.1 indicates serious multicollinearity
- TOL < 0.2 indicates potential problem

**4. Condition Number:**

```
κ = √(λₘₐₓ / λₘᵢₙ)
```

Where λₘₐₓ and λₘᵢₙ are largest and smallest eigenvalues of X'X.

Rules of thumb:
- κ < 10: No serious problem
- 10 < κ < 30: Moderate multicollinearity
- κ > 30: Severe multicollinearity

**5. Condition Index:**

```
CIⱼ = √(λₘₐₓ / λⱼ)
```

For each eigenvalue λⱼ.

**6. Eigenvalue Analysis:**

- Compute eigenvalues of X'X (after standardizing X)
- One or more eigenvalues close to zero indicates multicollinearity
- Number of small eigenvalues = number of near-linear dependencies

**7. Auxiliary Regressions:**

- Regress each X variable on all other X variables
- High R² values indicate multicollinearity
- Identifies which variables are involved

**8. Informal Detection:**

- Large changes in coefficients when variables added/removed
- Coefficients with wrong signs
- High R² but few significant t-ratios

**REMEDIAL MEASURES:**

| Method | Description |
|--------|-------------|
| Do Nothing | If prediction is the goal and multicollinearity will persist |
| Drop Variables | Remove redundant variables (may cause specification bias) |
| Combine Variables | Create indices or principal components |
| Ridge Regression | Biased but lower MSE estimation |
| Increase Sample Size | More data may reduce the problem |
| Prior Information | Use restricted least squares |
| Transform Variables | First differences, ratios |
| Principal Component Regression | Use orthogonal components |

### Question Set 7: Distributed Lag Models

**Q7.1:** "Explain distributed lag models. Derive the Koyck transformation and discuss problems in its estimation."

**Model Answer:**

**Introduction to Distributed Lag Models:**

In many economic relationships, the effect of an explanatory variable on the dependent variable is distributed over time. The dependent variable responds not just to current values but also to past values of the explanatory variable.

**General Distributed Lag Model:**

```
Yₜ = α + β₀Xₜ + β₁Xₜ₋₁ + β₂Xₜ₋₂ + ... + βₖXₜ₋ₖ + uₜ
```

Or with infinite lags:

```
Yₜ = α + Σᵢ₌₀^∞ βᵢXₜ₋ᵢ + uₜ
```

**Interpretation:**
- β₀ = Short-run (impact) multiplier
- Σβᵢ = Long-run (total) multiplier
- βᵢ = Lag weights showing how effect is distributed over time

**Problems with Unrestricted Estimation:**

1. Loss of degrees of freedom (many parameters)
2. Multicollinearity among lagged X values
3. Infinite lags cannot be estimated directly
4. Need to determine lag length k

**KOYCK TRANSFORMATION:**

**Assumption:**
The lag coefficients decline geometrically:

```
βᵢ = β₀λⁱ, where 0 < λ < 1
```

This implies:
- β₁ = β₀λ
- β₂ = β₀λ²
- β₃ = β₀λ³
- And so on...

**Derivation:**

**Step 1:** Write the infinite distributed lag model with geometric weights:

```
Yₜ = α + β₀Xₜ + β₀λXₜ₋₁ + β₀λ²Xₜ₋₂ + ... + uₜ
Yₜ = α + β₀(Xₜ + λXₜ₋₁ + λ²Xₜ₋₂ + ...) + uₜ ... (1)
```

**Step 2:** Lag equation (1) by one period:

```
Yₜ₋₁ = α + β₀(Xₜ₋₁ + λXₜ₋₂ + λ²Xₜ₋₃ + ...) + uₜ₋₁ ... (2)
```

**Step 3:** Multiply equation (2) by λ:

```
λYₜ₋₁ = λα + β₀(λXₜ₋₁ + λ²Xₜ₋₂ + λ³Xₜ₋₃ + ...) + λuₜ₋₁ ... (3)
```

**Step 4:** Subtract equation (3) from equation (1):

```
Yₜ - λYₜ₋₁ = α - λα + β₀Xₜ + uₜ - λuₜ₋₁
```

**Step 5:** Rearrange to get the Koyck Model:

```
Yₜ = α(1-λ) + β₀Xₜ + λYₜ₋₁ + vₜ
```

Where:
```
vₜ = uₜ - λuₜ₋₁
```

**Simplified Form:**

```
Yₜ = α* + β₀Xₜ + λYₜ₋₁ + vₜ
```

Where α* = α(1-λ)

**Interpretation of Parameters:**

| Parameter | Interpretation |
|-----------|----------------|
| β₀ | Short-run (impact) multiplier |
| λ | Speed of adjustment (0 < λ < 1) |
| β₀/(1-λ) | Long-run multiplier |
| α* | Transformed intercept |
| (1-λ) | Adjustment coefficient |

**Median Lag:**
```
Median Lag = -ln(2)/ln(λ)
```

**Mean Lag:**
```
Mean Lag = λ/(1-λ)
```

**PROBLEMS IN ESTIMATING KOYCK MODEL:**

**Problem 1: Stochastic Regressor**

The lagged dependent variable Yₜ₋₁ is stochastic (random), violating the classical assumption of non-stochastic regressors.

**Consequence:** OLS is still consistent if Yₜ₋₁ is independent of vₜ.

**Problem 2: Correlation Between Regressor and Error**

The error term vₜ = uₜ - λuₜ₋₁ is correlated with Yₜ₋₁.

**Proof:**
```
Yₜ₋₁ depends on uₜ₋₁ (from the original model)
vₜ contains -λuₜ₋₁
Therefore, Cov(Yₜ₋₁, vₜ) ≠ 0
```

**Consequence:** OLS is biased AND inconsistent.

**Problem 3: Autocorrelated Errors**

The transformed error vₜ = uₜ - λuₜ₋₁ follows an MA(1) process even if uₜ is white noise.

```
E(vₜ) = 0
Var(vₜ) = σ²ᵤ(1 + λ²)
Cov(vₜ, vₜ₋₁) = -λσ²ᵤ
Cov(vₜ, vₜ₋ₛ) = 0 for s > 1
```

**Consequence:**
- OLS standard errors are incorrect
- Durbin-Watson test is biased toward 2 (toward no autocorrelation)

**Problem 4: Durbin-Watson Test Invalid**

The Durbin-Watson test cannot be used because:
- There is a lagged dependent variable
- The test is biased toward accepting no autocorrelation

**ESTIMATION METHODS:**

**Method 1: Instrumental Variables (IV)**

- Use instruments that are correlated with Yₜ₋₁ but uncorrelated with vₜ
- Common instruments: Xₜ₋₁, Xₜ₋₂, Yₜ₋₂
- Provides consistent estimates

**Method 2: Maximum Likelihood**

- Specify the full likelihood function
- Account for MA(1) error structure
- Computationally intensive but efficient

**Method 3: Nonlinear Least Squares**

- Estimate original infinite lag model directly
- Impose geometric lag structure
- Iterate to convergence

**Method 4: Durbin's h-test for Autocorrelation**

Since D-W test is invalid, use Durbin's h-statistic:

```
h = ρ̂ × √(n / (1 - n × Var(λ̂)))
```

Where:
- ρ̂ = estimated first-order autocorrelation of residuals
- n = sample size
- Var(λ̂) = estimated variance of λ̂

Under H₀: h ~ N(0,1) asymptotically

**Limitations:**
- Cannot be computed if n × Var(λ̂) > 1
- Only valid for large samples

**Alternative: Breusch-Godfrey LM Test**

- Valid with lagged dependent variables
- Can test for higher-order autocorrelation
- Preferred in modern applications

### Question Set 8: 3SLS Estimation

**Q8.1:** "Explain the Three-Stage Least Squares (3SLS) estimation method. Compare it with 2SLS."

**Model Answer:**

**Introduction:**

Three-Stage Least Squares (3SLS) is a system method for estimating all equations of a simultaneous equations model jointly. It combines 2SLS with Generalized Least Squares (GLS) to account for cross-equation error correlations.

**Motivation:**

2SLS estimates each equation separately, ignoring:
1. Correlations between errors of different equations
2. Cross-equation restrictions

3SLS improves efficiency by using information from the entire system.

**THE THREE STAGES:**

**Stage 1: Obtain Instruments**

Same as first stage of 2SLS:
- Regress each endogenous variable on all exogenous variables in the system
- Obtain fitted values: Ŷ = X(X'X)⁻¹X'Y = PₓY

**Stage 2: Estimate Each Equation by 2SLS**

For each equation i:
```
yᵢ = Zᵢδᵢ + uᵢ
```

Obtain 2SLS estimates:
```
δ̂ᵢ,₂ₛₗₛ = (Ẑᵢ'Zᵢ)⁻¹Ẑᵢ'yᵢ
```

Calculate residuals:
```
êᵢ = yᵢ - Zᵢδ̂ᵢ,₂ₛₗₛ
```

Estimate the error covariance matrix:
```
σ̂ᵢⱼ = êᵢ'êⱼ / n
```

Form the estimated covariance matrix:
```
Σ̂ = [σ̂ᵢⱼ] (G × G matrix)
```

**Stage 3: Apply GLS to the System**

Stack all equations:
```
y = Zδ + u
```

Where:
```
y = [y₁]    Z = [Z₁  0   ...  0 ]    δ = [δ₁]    u = [u₁]
    [y₂]        [0   Z₂  ...  0 ]        [δ₂]        [u₂]
    [⋮ ]        [⋮   ⋮   ⋱   ⋮ ]        [⋮ ]        [⋮ ]
    [yG]        [0   0   ... ZG]        [δG]        [uG]
```

The covariance matrix of u is:
```
E(uu') = Σ ⊗ Iₙ
```

Where ⊗ denotes Kronecker product.

**3SLS Estimator:**

```
δ̂₃ₛₗₛ = [Ẑ'(Σ̂⁻¹ ⊗ Iₙ)Z]⁻¹ Ẑ'(Σ̂⁻¹ ⊗ Iₙ)y
```

Or equivalently:
```
δ̂₃ₛₗₛ = [Z'(Σ̂⁻¹ ⊗ Pₓ)Z]⁻¹ Z'(Σ̂⁻¹ ⊗ Pₓ)y
```

**Variance of 3SLS Estimator:**

```
Var(δ̂₃ₛₗₛ) = [Z'(Σ̂⁻¹ ⊗ Pₓ)Z]⁻¹
```

**COMPARISON: 2SLS vs 3SLS**

| Criterion | 2SLS | 3SLS |
|-----------|------|------|
| **Estimation Approach** | Single equation | System (all equations jointly) |
| **Information Used** | Only own equation | All equations |
| **Cross-equation Correlations** | Ignored | Utilized |
| **Consistency** | Yes | Yes |
| **Asymptotic Efficiency** | Less efficient | More efficient |
| **Computational Complexity** | Simpler | More complex |
| **Sensitivity to Misspecification** | Less sensitive | More sensitive |
| **When Σ is Diagonal** | 2SLS = 3SLS | 2SLS = 3SLS |
| **Finite Sample Properties** | May be better | May be worse |
| **Specification Errors** | Contained to one equation | Spread across system |

**When to Use 2SLS:**

1. When only one equation is of primary interest
2. When there is concern about misspecification in other equations
3. When cross-equation error correlations are small
4. When computational simplicity is desired
5. When sample size is small

**When to Use 3SLS:**

1. When all equations are correctly specified
2. When cross-equation error correlations are substantial
3. When maximum efficiency is desired
4. When the entire system is of interest
5. When sample size is large

**Special Cases:**

1. **If Σ = σ²I (no cross-equation correlation):**
   ```
   3SLS = 2SLS
   ```

2. **If all equations are exactly identified:**
   ```
   3SLS = 2SLS (asymptotically)
   ```

3. **If system is recursive with diagonal Σ:**
   ```
   OLS on each equation is consistent and efficient
   ```

**Properties of 3SLS:**

1. **Consistency:** 3SLS is consistent under standard assumptions
2. **Asymptotic Normality:** √n(δ̂₃ₛₗₛ - δ) → N(0, V) as n → ∞
3. **Asymptotic Efficiency:** 3SLS is asymptotically efficient among instrumental variable estimators
4. **Invariance:** 3SLS estimates are invariant to the normalization rule

### Derivation 1: OLS Estimator

**Model:** Y = Xβ + u

**Objective:** Minimize S = u'u = (Y - Xβ)'(Y - Xβ)

**Derivation:**

```
S = Y'Y - 2β'X'Y + β'X'Xβ

∂S/∂β = -2X'Y + 2X'Xβ = 0

X'Xβ = X'Y

β̂ = (X'X)⁻¹X'Y
```

### Derivation 2: Variance of OLS Estimator

```
β̂ = β + (X'X)⁻¹X'u

β̂ - β = (X'X)⁻¹X'u

Var(β̂) = E[(β̂ - β)(β̂ - β)']
        = E[(X'X)⁻¹X'uu'X(X'X)⁻¹]
        = (X'X)⁻¹X'E(uu')X(X'X)⁻¹
        = (X'X)⁻¹X'(σ²I)X(X'X)⁻¹
        = σ²(X'X)⁻¹
```

### Derivation 3: GLS Estimator

**Model:** Y = Xβ + u, where E(uu') = σ²Ω

**Transformation:** Ω = PP' (Cholesky decomposition)

Let P⁻¹Y = Y*, P⁻¹X = X*, P⁻¹u = u*

Then: Y* = X*β + u*, where E(u*u*') = σ²I

**GLS Estimator:**

```
β̂ᵍˡˢ = (X*'X*)⁻¹X*'Y*
      = (X'Ω⁻¹X)⁻¹X'Ω⁻¹Y
```

**Variance:**

```
Var(β̂ᵍˡˢ) = σ²(X'Ω⁻¹X)⁻¹
```

### Derivation 4: Ridge Regression Estimator

**Objective:** Minimize S = (Y - Xβ)'(Y - Xβ) + kβ'β

**Derivation:**

```
```
S = Y'Y - 2β'X'Y + β'X'Xβ + kβ'β

∂S/∂β = -2X'Y + 2X'Xβ + 2kβ = 0

(X'X + kI)β = X'Y

β̂ᵣᵢdgₑ = (X'X + kI)⁻¹X'Y
```

**Properties:**

1. **Bias:**
```
E(β̂ᵣᵢdgₑ) = (X'X + kI)⁻¹X'Xβ ≠ β (biased)
```

2. **Variance:**
```
Var(β̂ᵣᵢdgₑ) = σ²(X'X + kI)⁻¹X'X(X'X + kI)⁻¹
```

3. **MSE:**
```
MSE(β̂ᵣᵢdgₑ) = Var(β̂ᵣᵢdgₑ) + [Bias(β̂ᵣᵢdgₑ)]²
```

For appropriate k, MSE(β̂ᵣᵢdgₑ) < MSE(β̂ₒₗₛ)

## 7.2 Complete Derivation: Identification Conditions

### Order Condition Derivation

**Setup:**

Consider a system of G structural equations:

```
BY + ΓX = U
```

Where:
- B = G × G matrix of coefficients of endogenous variables
- Y = G × 1 vector of endogenous variables
- Γ = G × K matrix of coefficients of exogenous variables
- X = K × 1 vector of exogenous variables
- U = G × 1 vector of disturbances

**Reduced Form:**

```
Y = -B⁻¹ΓX + B⁻¹U = ΠX + V
```

Where Π = -B⁻¹Γ is the G × K matrix of reduced form coefficients.

**For the ith Equation:**

Let the ith structural equation be:
```
y₁ᵢ = -Σⱼ₌₂^gᵢ βᵢⱼyⱼ - Σₗ₌₁^kᵢ γᵢₗxₗ + uᵢ
```

Where:
- gᵢ = number of endogenous variables in equation i
- kᵢ = number of exogenous variables in equation i

**Identification Requirement:**

To identify the ith equation, we need to recover its gᵢ + kᵢ - 1 unknown parameters (excluding the normalized coefficient) from the reduced form.

**Available Information:**

From the reduced form, we have K reduced form coefficients for each endogenous variable in the equation.

**Counting Equations and Unknowns:**

Number of equations available = K - kᵢ (from excluded exogenous variables)
Number of unknowns = gᵢ - 1 (coefficients of other endogenous variables)

**Order Condition:**

For identification:
```
Number of equations ≥ Number of unknowns
K - kᵢ ≥ gᵢ - 1
```

**Classification:**
- K - kᵢ < gᵢ - 1: Under-identified
- K - kᵢ = gᵢ - 1: Exactly identified
- K - kᵢ > gᵢ - 1: Over-identified

### Rank Condition Derivation

**Why Order Condition is Not Sufficient:**

The order condition only counts the number of restrictions but doesn't ensure they are independent and effective.

**Rank Condition Statement:**

The ith equation is identified if and only if:
```
rank(Aᵢ) = G - 1
```

Where Aᵢ is the (G-1) × (K - kᵢ + G - gᵢ) matrix formed by:
- Taking coefficients of variables excluded from equation i
- From all other G - 1 equations

**Construction of Aᵢ:**

**Step 1:** Write the system in matrix form [B : Γ]

**Step 2:** Delete the ith row (equation being checked)

**Step 3:** Delete columns corresponding to variables included in equation i

**Step 4:** The remaining matrix is Aᵢ

**Step 5:** Check if rank(Aᵢ) = G - 1

**Example:**

Consider:
```
Equation 1: Y₁ + β₁₂Y₂ + γ₁₁X₁ = u₁
Equation 2: β₂₁Y₁ + Y₂ + γ₂₂X₂ = u₂
Equation 3: β₃₁Y₁ + β₃₂Y₂ + Y₃ + γ₃₃X₃ = u₃
```

**System Matrix [B : Γ]:**

```
         Y₁    Y₂    Y₃    X₁    X₂    X₃
Eq 1  [  1    β₁₂    0    γ₁₁    0     0  ]
Eq 2  [ β₂₁    1     0     0    γ₂₂    0  ]
Eq 3  [ β₃₁   β₃₂    1     0     0    γ₃₃ ]
```

**Check Identification of Equation 1:**

Variables in Equation 1: Y₁, Y₂, X₁
Variables excluded: Y₃, X₂, X₃

Matrix A₁ (from Equations 2 and 3, columns Y₃, X₂, X₃):
```
         Y₃    X₂    X₃
Eq 2  [  0    γ₂₂    0  ]
Eq 3  [  1     0    γ₃₃ ]
```

rank(A₁) = 2 = G - 1 ✓

**Equation 1 is identified.**

## 7.3 Complete Derivation: Durbin-Watson Test

### Detailed Derivation

**Model:**
```
Yₜ = Xₜβ + uₜ
```

**Assumption under H₁:**
```
uₜ = ρuₜ₋₁ + εₜ, where |ρ| < 1 and εₜ ~ iid(0, σ²ε)
```

**Hypotheses:**
```
H₀: ρ = 0 (no autocorrelation)
H₁: ρ > 0 (positive autocorrelation)
```

**Test Statistic:**

```
d = Σₜ₌₂ⁿ (eₜ - eₜ₋₁)² / Σₜ₌₁ⁿ eₜ²
```

Where eₜ = Yₜ - Ŷₜ are OLS residuals.

**Derivation of Relationship with ρ:**

**Step 1:** Expand the numerator
```
Σ(eₜ - eₜ₋₁)² = Σeₜ² - 2Σeₜeₜ₋₁ + Σeₜ₋₁²
```

**Step 2:** For large n, Σeₜ² ≈ Σeₜ₋₁²

Therefore:
```
Σ(eₜ - eₜ₋₁)² ≈ 2Σeₜ² - 2Σeₜeₜ₋₁
              = 2Σeₜ²[1 - Σeₜeₜ₋₁/Σeₜ²]
```

**Step 3:** Define the estimated autocorrelation coefficient
```
ρ̂ = Σeₜeₜ₋₁ / Σeₜ²
```

**Step 4:** Substitute into d
```
d = [2Σeₜ² - 2Σeₜeₜ₋₁] / Σeₜ²
  = 2[1 - Σeₜeₜ₋₁/Σeₜ²]
  = 2(1 - ρ̂)
```

**Relationship:**
```
d ≈ 2(1 - ρ̂)
```

**Solving for ρ̂:**
```
ρ̂ ≈ 1 - d/2
```

**Range of d:**

| Value of ρ | Value of d |
|------------|------------|
| ρ = +1 | d = 0 |
| ρ = 0 | d = 2 |
| ρ = -1 | d = 4 |

Therefore: 0 ≤ d ≤ 4

**Matrix Form of d:**

```
d = e'Ae / e'e
```

Where A is an n × n matrix:
```
A = [  1   -1    0    0   ...   0    0  ]
    [ -1    2   -1    0   ...   0    0  ]
    [  0   -1    2   -1   ...   0    0  ]
    [  ⋮    ⋮    ⋮    ⋮    ⋱    ⋮    ⋮  ]
    [  0    0    0    0   ...   2   -1  ]
    [  0    0    0    0   ...  -1    1  ]
```

**Distribution of d:**

Under H₀, d can be written as:
```
d = e'Ae / e'e = u'MAMu / u'Mu
```

Where M = I - X(X'X)⁻¹X' is the residual maker matrix.

The exact distribution depends on X, making it difficult to tabulate.

**Durbin-Watson Bounds:**

Durbin and Watson showed that:
```
dL ≤ d ≤ dU
```

Where dL and dU are bounds that depend only on n and k, not on X.

**Critical Region:**

For testing H₀: ρ = 0 vs H₁: ρ > 0 at significance level α:

| Condition | Decision |
|-----------|----------|
| d < dL | Reject H₀ (positive autocorrelation) |
| d > dU | Do not reject H₀ |
| dL ≤ d ≤ dU | Inconclusive |

For testing H₀: ρ = 0 vs H₁: ρ < 0:

| Condition | Decision |
|-----------|----------|
| d > 4 - dL | Reject H₀ (negative autocorrelation) |
| d < 4 - dU | Do not reject H₀ |
| 4 - dU ≤ d ≤ 4 - dL | Inconclusive |

# CHAPTER 8: NUMERICAL PROBLEM PATTERNS

## 8.1 Identification Problems

### Pattern 1: Basic Identification Check

**Problem Type:**
Given a system of simultaneous equations, check the identification status of each equation using order and rank conditions.

**Example Problem:**

Consider the following model:
```
Demand: Qᵈ = α₀ + α₁P + α₂Y + u₁
Supply: Qˢ = β₀ + β₁P + β₂R + β₃T + u₂
Equilibrium: Qᵈ = Qˢ = Q
```

Where:
- Q = quantity
- P = price
- Y = income (exogenous)
- R = rainfall (exogenous)
- T = technology index (exogenous)

Check the identification of demand and supply equations.

**Solution:**

**Step 1: Identify Variables**

Endogenous variables (G = 2): Q, P
Exogenous variables (K = 3): Y, R, T

**Step 2: Rewrite in Standard Form**

```
Demand: Q - α₁P = α₀ + α₂Y + u₁
Supply: Q - β₁P = β₀ + β₂R + β₃T + u₂
```

**Step 3: Apply Order Condition**

**For Demand Equation:**
- Endogenous variables in equation (g₁) = 2 (Q, P)
- Exogenous variables in equation (k₁) = 1 (Y)
- Excluded exogenous variables = K - k₁ = 3 - 1 = 2 (R, T)

Order condition: K - k₁ ≥ g₁ - 1
```
2 ≥ 2 - 1
2 ≥ 1 ✓
```

Since 2 > 1, demand equation is **over-identified**.

**For Supply Equation:**
- Endogenous variables in equation (g₂) = 2 (Q, P)
- Exogenous variables in equation (k₂) = 2 (R, T)
- Excluded exogenous variables = K - k₂ = 3 - 2 = 1 (Y)

Order condition: K - k₂ ≥ g₂ - 1
```
1 ≥ 2 - 1
1 ≥ 1 ✓
```

Since 1 = 1, supply equation is **exactly identified**.

**Step 4: Apply Rank Condition**

**System Matrix [B : Γ]:**

```
              Q      P      Y      R      T
Demand    [  1    -α₁    -α₂     0      0  ]
Supply    [  1    -β₁     0    -β₂    -β₃  ]
```

**For Demand Equation:**

Excluded variables: R, T

Matrix A₁ (coefficients of R, T in Supply equation):
```
A₁ = [-β₂  -β₃]
```

rank(A₁) = 1 (assuming β₂ ≠ 0 or β₃ ≠ 0)
Required rank = G - 1 = 2 - 1 = 1 ✓

**Demand equation is identified (over-identified).**

**For Supply Equation:**

Excluded variables: Y

Matrix A₂ (coefficients of Y in Demand equation):
```
A₂ = [-α₂]
```

rank(A₂) = 1 (assuming α₂ ≠ 0)
Required rank = G - 1 = 1 ✓

**Supply equation is identified (exactly identified).**

**Summary:**

| Equation | Order Condition | Rank Condition | Status |
|----------|-----------------|----------------|--------|
| Demand | 2 > 1 | rank = 1 | Over-identified |
| Supply | 1 = 1 | rank = 1 | Exactly identified |

**Estimation Methods:**
- Demand: 2SLS recommended
- Supply: ILS or 2SLS

## 8.2 OLS Estimation Problems

### Pattern 1: Basic OLS Calculation

**Example Problem:**

Given the following data:

| Observation | Y | X |
|-------------|---|---|
| 1 | 2 | 1 |
| 2 | 4 | 2 |
| 3 | 5 | 3 |
| 4 | 4 | 4 |
| 5 | 5 | 5 |

(a) Estimate the regression Y = β₀ + β₁X + u using OLS
(b) Calculate R²
(c) Test the significance of β₁ at 5% level
(d) Compute the Durbin-Watson statistic

**Solution:**

**Part (a): OLS Estimation**

**Step 1: Calculate Summary Statistics**

| Obs | Y | X | XY | X² | Y² |
|-----|---|---|----|----|-----|
| 1 | 2 | 1 | 2 | 1 | 4 |
| 2 | 4 | 2 | 8 | 4 | 16 |
| 3 | 5 | 3 | 15 | 9 | 25 |
| 4 | 4 | 4 | 16 | 16 | 16 |
| 5 | 5 | 5 | 25 | 25 | 25 |
| **Sum** | **20** | **15** | **66** | **55** | **86** |

n = 5
ΣY = 20, ΣX = 15
ΣXY = 66, ΣX² = 55, ΣY² = 86
Ȳ = 20/5 = 4, X̄ = 15/5 = 3

**Step 2: Calculate β̂₁**

```
β̂₁ = [ΣXY - nX̄Ȳ] / [ΣX² - nX̄²]
   = [66 - 5(3)(4)] / [55 - 5(9)]
   = [66 - 60] / [55 - 45]
   = 6/10
   = 0.6
```

**Step 3: Calculate β̂₀**

```
β̂₀ = Ȳ - β̂₁X̄
   = 4 - 0.6(3)
   = 4 - 1.8
   = 2.2
```

**Estimated Model: Ŷ = 2.2 + 0.6X**

**Part (b): Calculate R²**

**Step 1: Calculate TSS, ESS, RSS**

| Obs | Y | Ŷ | e = Y - Ŷ | e² | (Y - Ȳ)² | (Ŷ - Ȳ)² |
|-----|---|-----|-----------|-----|----------|----------|
| 1 | 2 | 2.8 | -0.8 | 0.64 | 4 | 1.44 |
| 2 | 4 | 3.4 | 0.6 | 0.36 | 0 | 0.36 |
| 3 | 5 | 4.0 | 1.0 | 1.00 | 1 | 0 |
| 4 | 4 | 4.6 | -0.6 | 0.36 | 0 | 0.36 |
| 5 | 5 | 5.2 | -0.2 | 0.04 | 1 | 1.44 |
| **Sum** | | | | **2.40** | **6** | **3.60** |

TSS = Σ(Y - Ȳ)² = 6
ESS = Σ(Ŷ - Ȳ)² = 3.60
RSS = Σe² = 2.40

**Verification:** TSS = ESS + RSS → 6 = 3.60 + 2.40 ✓

**Step 2: Calculate R²**

```
R² = ESS/TSS = 3.60/6 = 0.60
```

Or equivalently:
```
R² = 1 - RSS/TSS = 1 - 2.40/6 = 1 - 0.40 = 0.60
```

**Interpretation:** 60% of the variation in Y is explained by X.

**Part (c): Test Significance of β₁**

**Step 1: Calculate σ̂²**

```
σ̂² = RSS/(n - k) = 2.40/(5 - 2) = 2.40/3 = 0.80
σ̂ = √0.80 = 0.894
```

**Step 2: Calculate Var(β̂₁)**

```
Var(β̂₁) = σ̂² / Σ(X - X̄)²
        = 0.80 / 10
        = 0.08

SE(β̂₁) = √0.08 = 0.283
```

**Step 3: Calculate t-statistic**

```
H₀: β₁ = 0
H₁: β₁ ≠ 0

t = β̂₁ / SE(β̂₁) = 0.6 / 0.283 = 2.12
```

**Step 4: Compare with Critical Value**

Degrees of freedom = n - k = 5 - 2 = 3
t₀.₀₂₅,₃ = 3.182 (two-tailed, 5% level)

Since |t| = 2.12 < 3.182, we **fail to reject H₀**.

**Conclusion:** β₁ is not statistically significant at 5% level.

**Part (d): Durbin-Watson Statistic**

| Obs | eₜ | eₜ - eₜ₋₁ | (eₜ - eₜ₋₁)² | eₜ² |
|-----|------|-----------|--------------|------|
| 1 | -0.8 | - | - | 0.64 |
| 2 | 0.6 | 1.4 | 1.96 | 0.36 |
| 3 | 1.0 | 0.4 | 0.16 | 1.00 |
| 4 | -0.6 | -1.6 | 2.56 | 0.36 |
| 5 | -0.2 | 0.4 | 0.16 | 0.04 |
| **Sum** | | | **4.84** | **2.40** |

```
d = Σ(eₜ - eₜ₋₁)² / Σeₜ²
  = 4.84 / 2.40
  = 2.017
```

**Interpretation:**

Since d ≈ 2, this suggests no significant autocorrelation.

For n = 5, k = 1 (excluding intercept):
- dL ≈ 0.61
- dU ≈ 1.40

Since d = 2.017 > dU = 1.40, we conclude **no positive autocorrelation**.

## 8.3 Heteroscedasticity Testing Problems

### Pattern: Goldfeld-Quandt Test

**Example Problem:**

A researcher estimates the consumption function C = β₀ + β₁Y + u using 30 observations ordered by income Y. The data is divided into three groups of 10 observations each, with the middle group omitted. The RSS from the first 10 observations (low income) is 120, and the RSS from the last 10 observations (high income) is 450.

Test for heteroscedasticity using the Goldfeld-Quandt test at 5% significance level.

**Solution:**

**Step 1: State Hypotheses**

```
H₀: σ₁² = σ₂² (Homoscedasticity)
H₁: σ₁² ≠ σ₂² (Heteroscedasticity)
```

**Step 2: Calculate Test Statistic**

```
F = RSS₂ / RSS₁ (larger RSS in numerator)
  = 450 / 120
  = 3.75
```

**Step 3: Determine Degrees of Freedom**

For each sub-sample:
- n₁ = n₂ = 10
- k = 2 (including intercept)
- df = n - k = 10 - 2 = 8

```
F ~ F(8, 8) under H₀
```

**Step 4: Find Critical Value**

F₀.₀₂₅,₈,₈ = 4.43 (for two-tailed test at 5% level)

Or for one-tailed test:
F₀.₀₅,₈,₈ = 3.44

**Step 5: Decision**

Since F = 3.75 > F₀.₀₅,₈,₈ = 3.44 (one-tailed)

**Reject H₀ at 5% significance level.**

**Conclusion:** There is evidence of heteroscedasticity. The variance of the error term increases with income, which is common in consumption functions.

---

### Pattern: Breusch-Pagan Test

**Example Problem:**

A researcher estimates Y = β₀ + β₁X₁ + β₂X₂ + u using 50 observations. After obtaining residuals, an auxiliary regression of ê²/σ̂² on X₁ and X₂ yields an ESS of 8.5.

Test for heteroscedasticity using the Breusch-Pagan test at 5% level.

**Solution:**

**Step 1: State Hypotheses**

```
H₀: Homoscedasticity (Var(u) = σ² for all observations)
H₁: Heteroscedasticity (Var(u) = f(X₁, X₂))
```

**Step 2: Calculate Test Statistic**

```
LM = (1/2) × ESS from auxiliary regression
   = (1/2) × 8.5
   = 4.25
```

**Step 3: Determine Distribution**

Under H₀:
```
LM ~ χ²(p)
```

Where p = number of regressors in auxiliary regression = 2

**Step 4: Find Critical Value**

```
χ²₀.₀₅,₂ = 5.99
```

**Step 5: Decision**

Since LM = 4.25 < χ²₀.₀₅,₂ = 5.99

**Fail to reject H₀ at 5% significance level.**

**Conclusion:** There is no significant evidence of heteroscedasticity at the 5% level.

---

### Pattern: White's Test

**Example Problem:**

A researcher estimates Y = β₀ + β₁X + u using 100 observations. An auxiliary regression of ê² on a constant, X, and X² yields R² = 0.085.

Test for heteroscedasticity using White's test at 5% level.

**Solution:**

**Step 1: State Hypotheses**

```
H₀: Homoscedasticity
H₁: Heteroscedasticity
```

**Step 2: Calculate Test Statistic**

```
W = nR² = 100 × 0.085 = 8.5
```

**Step 3: Determine Distribution**

Under H₀:
```
W ~ χ²(q)
```

Where q = number of regressors in auxiliary regression (excluding constant) = 2

**Step 4: Find Critical Value**

```
χ²₀.₀₅,₂ = 5.99
```

**Step 5: Decision**

Since W = 8.5 > χ²₀.₀₅,₂ = 5.99

**Reject H₀ at 5% significance level.**

**Conclusion:** There is significant evidence of heteroscedasticity. The researcher should consider using heteroscedasticity-consistent standard errors or weighted least squares.

## 8.4 2SLS Estimation Problems

### Pattern: Complete 2SLS Calculation

**Example Problem:**

Consider the following supply-demand model:

```
Demand: Q = α₀ + α₁P + α₂Y + u₁
Supply: Q = β₀ + β₁P + u₂
```

Where Q = quantity, P = price (endogenous), Y = income (exogenous)

Given the following data (5 observations):

| Obs | Q | P | Y |
|-----|---|---|---|
| 1 | 10 | 5 | 20 |
| 2 | 15 | 8 | 25 |
| 3 | 12 | 6 | 22 |
| 4 | 18 | 10 | 30 |
| 5 | 20 | 12 | 35 |

Estimate the supply equation using 2SLS.

**Solution:**

**Step 1: Check Identification**

Supply equation:
- Endogenous variables: Q, P (g = 2)
- Exogenous variables in equation: none (k = 0)
- Total exogenous variables: Y (K = 1)

Order condition: K - k ≥ g - 1
```
1 - 0 ≥ 2 - 1
1 ≥ 1 ✓ (Exactly identified)
```

**Step 2: Stage 1 - Regress P on Y**

| Obs | P | Y | PY | Y² |
|-----|---|---|----|----|
| 1 | 5 | 20 | 100 | 400 |
| 2 | 8 | 25 | 200 | 625 |
| 3 | 6 | 22 | 132 | 484 |
| 4 | 10 | 30 | 300 | 900 |
| 5 | 12 | 35 | 420 | 1225 |
| **Sum** | **41** | **132** | **1152** | **3634** |

```
n = 5
ΣP = 41, ΣY = 132
ΣPY = 1152, ΣY² = 3634
P̄ = 8.2, Ȳ = 26.4
```

**Calculate regression coefficients:**

```
δ̂₁ = [ΣPY - nP̄Ȳ] / [ΣY² - nȲ²]
   = [1152 - 5(8.2)(26.4)] / [3634 - 5(26.4)²]
   = [1152 - 1082.4] / [3634 - 3484.8]
   = 69.6 / 149.2
   = 0.4665

δ̂₀ = P̄ - δ̂₁Ȳ
   = 8.2 - 0.4665(26.4)
   = 8.2 - 12.32
   = -4.12
```

**First Stage Regression: P̂ = -4.12 + 0.4665Y**

**Calculate Fitted Values P̂:**

| Obs | Y | P̂ = -4.12 + 0.4665Y |
|-----|---|----------------------|
| 1 | 20 | 5.21 |
| 2 | 25 | 7.54 |
| 3 | 22 | 6.14 |
| 4 | 30 | 9.88 |
| 5 | 35 | 12.21 |

**Step 3: Stage 2 - Regress Q on P̂**

| Obs | Q | P̂ | QP̂ | P̂² |
|-----|---|------|------|------|
| 1 | 10 | 5.21 | 52.1 | 27.14 |
| 2 | 15 | 7.54 | 113.1 | 56.85 |
| 3 | 12 | 6.14 | 73.68 | 37.70 |
| 4 | 18 | 9.88 | 177.84 | 97.61 |
| 5 | 20 | 12.21 | 244.2 | 149.08 |
| **Sum** | **75** | **40.98** | **660.92** | **368.38** |

```
ΣQ = 75, ΣP̂ = 40.98
ΣQP̂ = 660.92, ΣP̂² = 368.38
Q̄ = 15, P̂̄ = 8.196
```

**Calculate 2SLS coefficients:**

```
β̂₁ = [ΣQP̂ - nQ̄P̂̄] / [ΣP̂² - nP̂̄²]
   = [660.92 - 5(15)(8.196)] / [368.38 - 5(8.196)²]
   = [660.92 - 614.7] / [368.38 - 336.03]
   = 46.22 / 32.35
   = 1.429

β̂₀ = Q̄ - β̂₁P̂̄
   = 15 - 1.429(8.196)
   = 15 - 11.71
   = 3.29
```

**2SLS Estimated Supply Equation:**

```
Q̂ = 3.29 + 1.429P
```
```
**Interpretation:**
- β̂₀ = 3.29: When price is zero, quantity supplied is 3.29 units
- β̂₁ = 1.429: A one-unit increase in price leads to a 1.429 unit increase in quantity supplied

**Note:** The 2SLS estimate is consistent, unlike OLS which would be inconsistent due to the simultaneity between Q and P.

## 12.2 Model Solutions

### Solution to Question 1

**(a) Assumptions of Classical Linear Regression Model (5 marks)**

The Classical Linear Regression Model (CLRM) is based on the following assumptions:

**Assumption 1: Linearity**
The model is linear in parameters:
```
Y = Xβ + u
```
Where Y is n×1, X is n×k, β is k×1, and u is n×1.

**Assumption 2: Full Rank**
The matrix X has full column rank, i.e., rank(X) = k.
This ensures (X'X) is invertible.

**Assumption 3: Exogeneity**
The error term has zero conditional mean:
```
E(u|X) = 0
```
This implies E(u) = 0 and Cov(X, u) = 0.

**Assumption 4: Homoscedasticity**
The variance of error term is constant:
```
Var(uᵢ|X) = σ² for all i
```

**Assumption 5: No Autocorrelation**
Error terms are uncorrelated:
```
Cov(uᵢ, uⱼ|X) = 0 for all i ≠ j
```

Assumptions 4 and 5 together imply:
```
Var(u|X) = σ²Iₙ (Spherical disturbances)
```

**Assumption 6: Non-stochastic X (Optional)**
X is either fixed in repeated samples or independent of u.

**Assumption 7: Normality (For inference)**
```
u ~ N(0, σ²Iₙ)
```
Note: This is not required for Gauss-Markov theorem.

**(b) Proof that OLS is BLUE (10 marks)**

**OLS Estimator:**
```
β̂ = (X'X)⁻¹X'Y
```

**To Prove:** β̂ is Best Linear Unbiased Estimator (BLUE)

**Part 1: Linearity (1 mark)**

β̂ = (X'X)⁻¹X'Y = CY

Where C = (X'X)⁻¹X' is a k×n matrix of constants.

Therefore, β̂ is a linear function of Y. ✓

**Part 2: Unbiasedness (3 marks)**

Substituting Y = Xβ + u:
```
β̂ = (X'X)⁻¹X'Y
   = (X'X)⁻¹X'(Xβ + u)
   = (X'X)⁻¹X'Xβ + (X'X)⁻¹X'u
   = Iₖβ + (X'X)⁻¹X'u
   = β + (X'X)⁻¹X'u
```

Taking expectation:
```
E(β̂) = E[β + (X'X)⁻¹X'u]
     = β + (X'X)⁻¹X'E(u)
     = β + (X'X)⁻¹X' · 0
     = β
```

Therefore, β̂ is unbiased. ✓

**Part 3: Best (Minimum Variance) (6 marks)**

**Step 1:** Find variance of OLS estimator
```
β̂ - β = (X'X)⁻¹X'u

Var(β̂) = E[(β̂ - β)(β̂ - β)']
       = E[(X'X)⁻¹X'u · u'X(X'X)⁻¹]
       = (X'X)⁻¹X' E(uu') X(X'X)⁻¹
       = (X'X)⁻¹X' (σ²Iₙ) X(X'X)⁻¹
       = σ²(X'X)⁻¹X'X(X'X)⁻¹
       = σ²(X'X)⁻¹
```

**Step 2:** Consider any other linear unbiased estimator

Let β̃ = AY be another linear unbiased estimator, where A is a k×n matrix.

For unbiasedness:
```
E(β̃) = E(AY) = E[A(Xβ + u)] = AXβ + AE(u) = AXβ
```

For E(β̃) = β for all β, we need:
```
AX = Iₖ
```

**Step 3:** Express A in terms of OLS

Write A = (X'X)⁻¹X' + D, where D is a k×n matrix.

For AX = Iₖ:
```
[(X'X)⁻¹X' + D]X = Iₖ
(X'X)⁻¹X'X + DX = Iₖ
Iₖ + DX = Iₖ
DX = 0
```

**Step 4:** Find variance of β̃
```
Var(β̃) = Var(AY)
       = A · Var(Y) · A'
       = A(σ²Iₙ)A'
       = σ²AA'
```

Substituting A = (X'X)⁻¹X' + D:
```
AA' = [(X'X)⁻¹X' + D][(X'X)⁻¹X' + D]'
    = [(X'X)⁻¹X' + D][X(X'X)⁻¹ + D']
    = (X'X)⁻¹X'X(X'X)⁻¹ + (X'X)⁻¹X'D' + DX(X'X)⁻¹ + DD'
    = (X'X)⁻¹ + 0 + 0 + DD'    [since DX = 0, so X'D' = 0]
    = (X'X)⁻¹ + DD'
```

Therefore:
```
Var(β̃) = σ²[(X'X)⁻¹ + DD']
       = σ²(X'X)⁻¹ + σ²DD'
       = Var(β̂) + σ²DD'
```

**Step 5:** Compare variances

Since DD' is a positive semi-definite matrix (for any matrix D, DD' ≥ 0):
```
σ²DD' ≥ 0
```

Therefore:
```
Var(β̃) = Var(β̂) + σ²DD' ≥ Var(β̂)
```

The equality holds only when D = 0, i.e., when β̃ = β̂.

**Conclusion:**

The OLS estimator β̂ has the minimum variance among all linear unbiased estimators.

Therefore, β̂ is BLUE (Best Linear Unbiased Estimator). ∎

### Solution to Question 2

**(a) Heteroscedasticity and its Consequences (7 marks)**

**Definition (2 marks):**

Heteroscedasticity refers to the situation where the variance of the error term is not constant across observations:
```
Var(uᵢ) = σᵢ² ≠ σ² for all i
```

In matrix notation:
```
Var(u) = Ω = diag(σ₁², σ₂², ..., σₙ²) ≠ σ²Iₙ
```

**Common Causes:**
- Error learning (variance decreases over time)
- Income-expenditure data (variance increases with income)
- Improved data collection techniques
- Presence of outliers
- Model misspecification

**Consequences on OLS Estimators (5 marks):**

| Property | Effect |
|----------|--------|
| **Unbiasedness** | OLS estimators remain unbiased. E(β̂) = β still holds because unbiasedness does not depend on the variance structure. |
| **Consistency** | OLS estimators remain consistent. |
| **Efficiency** | OLS is no longer BLUE. It is no longer the minimum variance estimator. GLS becomes more efficient. |
| **Variance Estimation** | The usual formula Var(β̂) = σ²(X'X)⁻¹ is incorrect. The true variance is Var(β̂) = (X'X)⁻¹X'ΩX(X'X)⁻¹ |
| **Standard Errors** | Estimated standard errors are biased (usually underestimated when heteroscedasticity is related to regressors). |
| **t and F Tests** | Test statistics are invalid. Confidence intervals and hypothesis tests are unreliable. |
| **R²** | R² is not affected but its interpretation may be misleading. |

**Summary:** OLS remains unbiased but is inefficient, and all inference procedures become invalid.

**(b) Goldfeld-Quandt Test (8 marks)**

**Purpose:** To test for heteroscedasticity when the variance is suspected to be related to one of the explanatory variables.

**Assumptions:**
- The error terms are normally distributed
- The variance is monotonically related to a known variable Z

**Hypotheses:**
```
H₀: σ₁² = σ₂² = ... = σₙ² (Homoscedasticity)
H₁: σᵢ² increases (or decreases) with Zᵢ (Heteroscedasticity)
```

**Procedure:**

**Step 1: Order Observations**
Arrange all n observations in ascending (or descending) order according to the variable Z suspected of causing heteroscedasticity.

**Step 2: Divide the Sample**
- Omit c central observations (typically c ≈ n/3)
- Divide remaining observations into two groups:
  - Group 1: First (n-c)/2 observations (low Z values)
  - Group 2: Last (n-c)/2 observations (high Z values)

**Step 3: Run Separate Regressions**
- Estimate the regression model separately for each group
- Obtain RSS₁ from Group 1 (expected lower variance)
- Obtain RSS₂ from Group 2 (expected higher variance)

**Step 4: Calculate Test Statistic**
```
F = RSS₂/df₂ ÷ RSS₁/df₁ = RSS₂/RSS₁ (if df₁ = df₂)
```

Where:
- df₁ = df₂ = (n-c)/2 - k
- k = number of parameters (including intercept)

**Step 5: Decision Rule**

Under H₀:
```
F ~ F(df₂, df₁)
```

- If F > Fα,df₂,df₁: Reject H₀ (heteroscedasticity present)
- If F ≤ Fα,df₂,df₁: Do not reject H₀

**Advantages:**
- Simple to implement
- Does not require specification of heteroscedasticity form
- Powerful when variance is monotonically related to Z

**Limitations:**
- Requires knowledge of which variable causes heteroscedasticity
- Loss of observations (central c observations omitted)
- Assumes monotonic relationship
- May not detect non-monotonic heteroscedasticity
- Sensitive to choice of c

**Example:**

If n = 30, c = 10, k = 2:
- Each group has (30-10)/2 = 10 observations
- df₁ = df₂ = 10 - 2 = 8
- If RSS₁ = 100 and RSS₂ = 350
- F = 350/100 = 3.5
- Compare with F₀.₀₅,₈,₈ = 3.44
- Since 3.5 > 3.44, reject H₀ at 5% level

### Solution to Question 3

**Comparison of 2SLS and 3SLS (10 marks)**

**Introduction:**

Both 2SLS (Two-Stage Least Squares) and 3SLS (Three-Stage Least Squares) are instrumental variable methods for estimating simultaneous equations models. They differ in the amount of information used and their efficiency properties.

**Detailed Comparison:**

| Criterion | 2SLS | 3SLS |
|-----------|------|------|
| **Estimation Approach** | Single equation method; estimates each equation separately | System method; estimates all equations jointly |
| **Information Used** | Uses only information from the equation being estimated | Uses information from all equations in the system |
| **Cross-equation Correlations** | Ignores correlations between errors of different equations | Exploits cross-equation error correlations |
| **Stages** | Two stages: (1) Regress endogenous on all exogenous, (2) Use fitted values in structural equation | Three stages: (1) Same as 2SLS stage 1, (2) Estimate each equation by 2SLS to get error covariance matrix, (3) Apply GLS to the system |
| **Consistency** | Consistent | Consistent |
| **Asymptotic Efficiency** | Less efficient | More efficient (asymptotically) |
| **Finite Sample Properties** | May have better finite sample properties | May have worse finite sample properties |
| **Computational Complexity** | Simpler | More complex |
| **Sensitivity to Misspecification** | Less sensitive; errors in one equation don't affect others | More sensitive; misspecification in one equation affects all estimates |
| **When Σ is Diagonal** | 2SLS = 3SLS | 2SLS = 3SLS |
| **Over-identification** | Can handle | Can handle |

**Mathematical Comparison:**

**2SLS Estimator (for equation i):**
```
δ̂ᵢ,₂ₛₗₛ = (Ẑᵢ'Zᵢ)⁻¹Ẑᵢ'yᵢ
```

**3SLS Estimator (for entire system):**
```
δ̂₃ₛₗₛ = [Z'(Σ̂⁻¹ ⊗ Pₓ)Z]⁻¹ Z'(Σ̂⁻¹ ⊗ Pₓ)y
```

Where Σ̂ is the estimated error covariance matrix from 2SLS residuals.

**When to Prefer 2SLS:**

1. **Single equation of interest:** When only one equation needs to be estimated and others are not of primary concern.

2. **Specification uncertainty:** When there is doubt about the correct specification of other equations in the system.

3. **Robustness required:** When robustness to misspecification is more important than efficiency.

4. **Small sample size:** When the sample size is small and finite sample properties matter.

5. **Computational simplicity:** When computational resources are limited.

6. **Preliminary analysis:** For initial estimation before considering system methods.

**When to Prefer 3SLS:**

1. **All equations important:** When all equations in the system are of interest and correctly specified.

2. **High cross-equation correlations:** When errors across equations are substantially correlated.

3. **Maximum efficiency needed:** When obtaining the most efficient estimates is crucial.

4. **Large sample size:** When the sample is large enough for asymptotic properties to dominate.

5. **Cross-equation restrictions:** When there are restrictions linking parameters across equations.

6. **System-wide analysis:** When the entire system needs to be analyzed together.

**Special Cases:**

1. **If Σ = σ²I (no cross-equation correlation):**
   - 3SLS reduces to 2SLS
   - No efficiency gain from using 3SLS

2. **If all equations are exactly identified:**
   - 3SLS and 2SLS are asymptotically equivalent

3. **Recursive systems with diagonal Σ:**
   - OLS on each equation is consistent and efficient
   - Neither 2SLS nor 3SLS needed

**Conclusion:**

The choice between 2SLS and 3SLS depends on the trade-off between efficiency and robustness. 2SLS is preferred when robustness is important or when only specific equations are of interest. 3SLS is preferred when all equations are correctly specified and maximum efficiency is desired.

### Solution to Question 4

**(a) Derivation of Durbin-Watson Statistic (10 marks)**

**Model:**
```
Yₜ = Xₜβ + uₜ, t = 1, 2, ..., n
```

**Assumption under Alternative Hypothesis:**
The errors follow an AR(1) process:
```
uₜ = ρuₜ₋₁ + εₜ, |ρ| < 1
```
Where εₜ ~ iid(0, σ²ε)

**Hypotheses:**
```
H₀: ρ = 0 (No autocorrelation)
H₁: ρ ≠ 0 (Autocorrelation present)
```

**Definition of Durbin-Watson Statistic:**

```
d = Σₜ₌₂ⁿ (eₜ - eₜ₋₁)² / Σₜ₌₁ⁿ eₜ²
```

Where eₜ = Yₜ - Ŷₜ are OLS residuals.

**Derivation of Relationship with ρ:**

**Step 1:** Expand the numerator

```
Σₜ₌₂ⁿ (eₜ - eₜ₋₁)² = Σₜ₌₂ⁿ (eₜ² - 2eₜeₜ₋₁ + eₜ₋₁²)
                    = Σₜ₌₂ⁿ eₜ² - 2Σₜ₌₂ⁿ eₜeₜ₋₁ + Σₜ₌₂ⁿ eₜ₋₁²
```

**Step 2:** Approximate for large n

For large n:
```
Σₜ₌₂ⁿ eₜ² ≈ Σₜ₌₁ⁿ eₜ²
Σₜ₌₂ⁿ eₜ₋₁² ≈ Σₜ₌₁ⁿ eₜ²
```

Therefore:
```
Σₜ₌₂ⁿ (eₜ - eₜ₋₁)² ≈ 2Σₜ₌₁ⁿ eₜ² - 2Σₜ₌₂ⁿ eₜeₜ₋₁
```

**Step 3:** Substitute into d

```
d ≈ [2Σeₜ² - 2Σeₜeₜ₋₁] / Σeₜ²
  = 2 - 2(Σeₜeₜ₋₁ / Σeₜ²)
  = 2[1 - (Σeₜeₜ₋₁ / Σeₜ²)]
```

**Step 4:** Define estimated autocorrelation coefficient

```
ρ̂ = Σₜ₌₂ⁿ eₜeₜ₋₁ / Σₜ₌₁ⁿ eₜ²
```

**Step 5:** Establish relationship

```
d ≈ 2(1 - ρ̂)
```

Solving for ρ̂:
```
ρ̂ ≈ 1 - d/2
```

**Range of d:**

| Value of ρ | Value of d | Interpretation |
|------------|------------|----------------|
| ρ = +1 | d = 0 | Perfect positive autocorrelation |
| ρ = 0 | d = 2 | No autocorrelation |
| ρ = -1 | d = 4 | Perfect negative autocorrelation |

Therefore: **0 ≤ d ≤ 4**

**Matrix Representation:**

The Durbin-Watson statistic can be written as:
```
d = e'Ae / e'e
```

Where A is an n×n tridiagonal matrix:
```
A = | 1  -1   0   0  ...  0   0 |
    |-1   2  -1   0  ...  0   0 |
    | 0  -1   2  -1  ...  0   0 |
    | :   :   :   :   ⋱   :   : |
    | 0   0   0   0  ...  2  -1 |
    | 0   0   0   0  ... -1   1 |
```

**Decision Rules:**

For testing H₀: ρ = 0 vs H₁: ρ > 0:

| Region | Decision |
|--------|----------|
| d < dL | Reject H₀ (positive autocorrelation) |
| d > dU | Do not reject H₀ |
| dL ≤ d ≤ dU | Inconclusive |

For testing H₀: ρ = 0 vs H₁: ρ < 0:

| Region | Decision |
|--------|----------|
| d > 4 - dL | Reject H₀ (negative autocorrelation) |
| d < 4 - dU | Do not reject H₀ |
| 4 - dU ≤ d ≤ 4 - dL | Inconclusive |

---

**(b) Limitations of Durbin-Watson Test (5 marks)**

1. **Inconclusive Region:**
   - The test has a zone of indecision (dL ≤ d ≤ dU)
   - This region can be large, especially for small samples
   - No definitive conclusion can be drawn in this region

2. **Lagged Dependent Variable:**
   - The test is biased toward 2 (toward accepting H₀) when the model includes lagged dependent variable
   - In such cases, Durbin's h-test or Breusch-Godfrey test should be used

3. **Only Tests for AR(1):**
   - Only detects first-order autocorrelation
   - Cannot detect higher-order autocorrelation (AR(2), AR(3), etc.)
   - Cannot detect moving average errors

4. **Requires Specific Assumptions:**
   - Assumes non-stochastic regressors
   - Assumes normally distributed errors
   - Model must include an intercept term

5. **Bounds Test:**
   - Critical values depend on the X matrix
   - Only bounds (dL and dU) are tabulated, not exact critical values
   - May lead to inconclusive results

6. **One-Sided Test:**
   - Originally designed for testing positive autocorrelation
   - Testing for negative autocorrelation requires modification
   - Two-sided testing is less straightforward

7. **Sample Size Requirements:**
   - Tables typically available for n ≥ 15
   - May not be reliable for very small samples

8. **Assumes Stationarity:**
   - Assumes the autocorrelation structure is stable
   - May not detect time-varying autocorrelation

### Solution to Question 5

**Derivation of Koyck Model (15 marks)**

**Introduction:**

The Koyck distributed lag model provides a parsimonious way to estimate an infinite distributed lag model by imposing a geometric decay structure on the lag coefficients.

**Starting Point: Infinite Distributed Lag Model**

```
Yₜ = α + β₀Xₜ + β₁Xₜ₋₁ + β₂Xₜ₋₂ + β₃Xₜ₋₃ + ... + uₜ
   = α + Σᵢ₌₀^∞ βᵢXₜ₋ᵢ + uₜ ... (1)
```

**Problems with Direct Estimation:**
- Infinite number of parameters
- Loss of degrees of freedom
- Multicollinearity among lagged X values
- Cannot determine lag length

**Koyck's Assumption:**

The lag coefficients decline geometrically:
```
βᵢ = β₀λⁱ, where 0 < λ < 1
```

This implies:
- β₀ = β₀ (impact multiplier)
- β₁ = β₀λ
- β₂ = β₀λ²
- β₃ = β₀λ³
- And so on...

**Derivation:**

**Step 1:** Substitute geometric lag structure into equation (1)

```
Yₜ = α + β₀Xₜ + β₀λXₜ₋₁ + β₀λ²Xₜ₋₂ + β₀λ³Xₜ₋₃ + ... + uₜ
   = α + β₀(Xₜ + λXₜ₋₁ + λ²Xₜ₋₂ + λ³Xₜ₋₃ + ...) + uₜ ... (2)
```

**Step 2:** Lag equation (2) by one period

```
Yₜ₋₁ = α + β₀(Xₜ₋₁ + λXₜ₋₂ + λ²Xₜ₋₃ + λ³Xₜ₋₄ + ...) + uₜ₋₁ ... (3)
```

**Step 3:** Multiply equation (3) by λ

```
λYₜ₋₁ = λα + β₀(λXₜ₋₁ + λ²Xₜ₋₂ + λ³Xₜ₋₃ + λ⁴Xₜ₋₄ + ...) + λuₜ₋₁ ... (4)
```

**Step 4:** Subtract equation (4) from equation (2)

```
Yₜ - λYₜ₋₁ = α - λα + β₀Xₜ + β₀(λXₜ₋₁ + λ²Xₜ₋₂ + ...) 
              - β₀(λXₜ₋₁ + λ²X

```
Yₜ - λYₜ₋₁ = α - λα + β₀Xₜ + β₀(λXₜ₋₁ + λ²Xₜ₋₂ + ...) 
              - β₀(λXₜ₋₁ + λ²Xₜ₋₂ + ...) + uₜ - λuₜ₋₁
```

**Step 5:** Simplify

The terms β₀(λXₜ₋₁ + λ²Xₜ₋₂ + ...) cancel out:

```
Yₜ - λYₜ₋₁ = α(1 - λ) + β₀Xₜ + (uₜ - λuₜ₋₁)
```

**Step 6:** Rearrange to get the Koyck Model

```
Yₜ = α(1 - λ) + β₀Xₜ + λYₜ₋₁ + vₜ
```

Where:
```
vₜ = uₜ - λuₜ₋₁
```

**Final Koyck Model:**

```
Yₜ = α* + β₀Xₜ + λYₜ₋₁ + vₜ
```

Where:
- α* = α(1 - λ) = transformed intercept
- β₀ = short-run (impact) multiplier
- λ = rate of decay (0 < λ < 1)
- vₜ = uₜ - λuₜ₋₁ = transformed error term (MA(1) process)

**Interpretation of Parameters:**

| Parameter | Interpretation | Formula |
|-----------|----------------|---------|
| β₀ | Short-run multiplier | Direct effect of Xₜ on Yₜ |
| λ | Adjustment speed | Rate at which effects decay |
| β₀/(1-λ) | Long-run multiplier | Total cumulative effect |
| λ/(1-λ) | Mean lag | Average time for effect |
| -ln(2)/ln(λ) | Median lag | Time for half the effect |

**Example:**
If β₀ = 0.4 and λ = 0.6:
- Short-run multiplier = 0.4
- Long-run multiplier = 0.4/(1-0.6) = 0.4/0.4 = 1.0
- Mean lag = 0.6/(1-0.6) = 1.5 periods
- Median lag = -ln(2)/ln(0.6) = 1.36 periods

---

**Problems in Estimating the Koyck Model:**

**Problem 1: Stochastic Regressor**

The lagged dependent variable Yₜ₋₁ is a stochastic (random) variable, not a fixed regressor.

**Implications:**
- Violates the classical assumption of non-stochastic regressors
- OLS is still consistent if Yₜ₋₁ is independent of vₜ
- However, this independence does not hold in the Koyck model

**Problem 2: Correlation Between Yₜ₋₁ and vₜ**

This is the most serious problem. The error term vₜ = uₜ - λuₜ₋₁ is correlated with Yₜ₋₁.

**Proof:**

From the original model:
```
Yₜ₋₁ = α + β₀Xₜ₋₁ + β₀λXₜ₋₂ + ... + uₜ₋₁
```

So Yₜ₋₁ depends on uₜ₋₁.

The error term vₜ = uₜ - λuₜ₋₁ contains the term -λuₜ₋₁.

Therefore:
```
Cov(Yₜ₋₁, vₜ) = Cov(Yₜ₋₁, uₜ - λuₜ₋₁)
              = Cov(Yₜ₋₁, uₜ) - λCov(Yₜ₋₁, uₜ₋₁)
              = 0 - λVar(uₜ₋₁)    [since Yₜ₋₁ depends on uₜ₋₁]
              = -λσ²ᵤ ≠ 0
```

**Consequence:**
- OLS estimators are **biased**
- OLS estimators are **inconsistent**
- The bias does not disappear even with large samples

**Problem 3: Autocorrelated Errors**

Even if the original errors uₜ are white noise (uncorrelated), the transformed errors vₜ follow an MA(1) process.

**Properties of vₜ = uₜ - λuₜ₋₁:**

```
E(vₜ) = E(uₜ) - λE(uₜ₋₁) = 0

Var(vₜ) = Var(uₜ - λuₜ₋₁)
        = Var(uₜ) + λ²Var(uₜ₋₁)
        = σ²ᵤ(1 + λ²)

Cov(vₜ, vₜ₋₁) = E[(uₜ - λuₜ₋₁)(uₜ₋₁ - λuₜ₋₂)]
              = E[uₜuₜ₋₁ - λuₜuₜ₋₂ - λuₜ₋₁² + λ²uₜ₋₁uₜ₋₂]
              = 0 - 0 - λσ²ᵤ + 0
              = -λσ²ᵤ

Cov(vₜ, vₜ₋ₛ) = 0 for s > 1
```

**First-order autocorrelation coefficient:**
```
ρ₁ = Cov(vₜ, vₜ₋₁)/Var(vₜ) = -λσ²ᵤ / σ²ᵤ(1 + λ²) = -λ/(1 + λ²)
```

**Consequences:**
- Standard errors are incorrect
- t-tests and F-tests are invalid
- Confidence intervals are unreliable

**Problem 4: Durbin-Watson Test is Invalid**

The Durbin-Watson test cannot be used to test for autocorrelation because:
- The model contains a lagged dependent variable
- The D-W statistic is biased toward 2 (toward accepting no autocorrelation)
- This gives a false sense of no autocorrelation problem

**Problem 5: Identification of Original Parameters**

From the estimated Koyck model:
```
Yₜ = α* + β₀Xₜ + λYₜ₋₁ + vₜ
```

We can estimate α*, β₀, and λ directly.

To recover the original intercept α:
```
α = α*/(1 - λ)
```

This requires accurate estimation of both α* and λ.

---

**Solutions to Estimation Problems:**

**Solution 1: Instrumental Variables (IV) Method**

Use instruments that are:
- Correlated with Yₜ₋₁
- Uncorrelated with vₜ

**Common instruments:**
- Xₜ₋₁, Xₜ₋₂ (lagged exogenous variables)
- Yₜ₋₂ (further lagged dependent variable)

**Procedure:**
1. Regress Yₜ₋₁ on instruments to get Ŷₜ₋₁
2. Replace Yₜ₋₁ with Ŷₜ₋₁ in the Koyck model
3. Estimate by OLS

**Solution 2: Maximum Likelihood Estimation**

- Specify the complete likelihood function
- Account for the MA(1) error structure
- Maximize likelihood numerically

**Advantages:**
- Asymptotically efficient
- Accounts for error structure

**Disadvantages:**
- Computationally intensive
- Requires distributional assumptions

**Solution 3: Nonlinear Least Squares**

- Estimate the original infinite lag model directly
- Impose the geometric lag constraint
- Use iterative numerical methods

**Solution 4: Durbin's h-Test for Autocorrelation**

Since D-W test is invalid, use Durbin's h-statistic:

```
h = ρ̂ √(n / (1 - n·Var(λ̂)))
```

Where:
- ρ̂ = estimated first-order autocorrelation of residuals ≈ 1 - d/2
- n = sample size
- Var(λ̂) = estimated variance of λ̂

Under H₀ (no autocorrelation): h ~ N(0,1) asymptotically

**Limitations of h-test:**
- Cannot be computed if n·Var(λ̂) ≥ 1
- Only valid asymptotically

**Alternative: Breusch-Godfrey LM Test**

- Valid with lagged dependent variables
- Can test for higher-order autocorrelation
- Preferred in modern applications

### Solution to Question 6

**Simultaneous Equation Model Analysis (15 marks)**

**Given Model:**
```
C = α₀ + α₁Y + α₂C₋₁ + u₁    (Consumption function)
I = β₀ + β₁Y + β₂r + u₂       (Investment function)
Y = C + I + G                  (Income identity)
```

**(a) Identification of Variables (2 marks)**

**Endogenous Variables (Determined within the system):**
- C (Consumption)
- I (Investment)
- Y (Income)

**Number of Endogenous Variables: G = 3**

**Exogenous/Predetermined Variables (Determined outside the system):**
- G (Government expenditure) - policy variable
- r (Interest rate) - assumed exogenous
- C₋₁ (Lagged consumption) - predetermined

**Number of Exogenous Variables: K = 3**

---

**(b) Identification Status (10 marks)**

**Step 1: Rewrite Equations in Standard Form**

```
Equation 1 (Consumption): C - α₁Y = α₀ + α₂C₋₁ + u₁
Equation 2 (Investment):  I - β₁Y = β₀ + β₂r + u₂
Equation 3 (Identity):    Y - C - I = G
```

**Step 2: Construct the System Matrix**

Writing in matrix form [B : Γ]:

```
                    C      I      Y    |   Const   C₋₁    r      G
Consumption    [    1      0    -α₁   |   -α₀    -α₂     0      0   ]
Investment     [    0      1    -β₁   |   -β₀     0     -β₂     0   ]
Identity       [   -1     -1     1    |    0      0      0     -1   ]
```

**Step 3: Apply Order Condition**

**Order Condition Formula:**
```
K - kᵢ ≥ gᵢ - 1
```

Where:
- K = total exogenous variables = 3 (G, r, C₋₁)
- kᵢ = exogenous variables in equation i
- gᵢ = endogenous variables in equation i

**For Consumption Function:**
- Endogenous variables in equation: C, Y → gᵢ = 2
- Exogenous variables in equation: C₋₁ → kᵢ = 1
- Excluded exogenous variables: G, r → K - kᵢ = 2

Order Condition:
```
K - kᵢ ≥ gᵢ - 1
2 ≥ 2 - 1
2 ≥ 1 ✓
```

Since 2 > 1, **Consumption function is OVER-IDENTIFIED** by order condition.

**For Investment Function:**
- Endogenous variables in equation: I, Y → gᵢ = 2
- Exogenous variables in equation: r → kᵢ = 1
- Excluded exogenous variables: G, C₋₁ → K - kᵢ = 2

Order Condition:
```
K - kᵢ ≥ gᵢ - 1
2 ≥ 2 - 1
2 ≥ 1 ✓
```

Since 2 > 1, **Investment function is OVER-IDENTIFIED** by order condition.

**Step 4: Apply Rank Condition**

**Rank Condition:** The equation is identified if and only if rank of the matrix formed by coefficients of excluded variables in other equations equals G - 1 = 2.

**For Consumption Function:**

Excluded variables: G, r

Matrix A₁ (coefficients of G, r in other equations):

```
                    r       G
Investment    [   -β₂      0   ]
Identity      [    0      -1   ]
```

```
A₁ = | -β₂    0  |
     |  0    -1  |
```

Determinant = (-β₂)(-1) - (0)(0) = β₂

If β₂ ≠ 0, then rank(A₁) = 2 = G - 1 ✓

**Consumption function is IDENTIFIED (Over-identified)** assuming β₂ ≠ 0.

**For Investment Function:**

Excluded variables: G, C₋₁

Matrix A₂ (coefficients of G, C₋₁ in other equations):

```
                   C₋₁      G
Consumption   [   -α₂      0   ]
Identity      [    0      -1   ]
```

```
A₂ = | -α₂    0  |
     |  0    -1  |
```

Determinant = (-α₂)(-1) - (0)(0) = α₂

If α₂ ≠ 0, then rank(A₂) = 2 = G - 1 ✓

**Investment function is IDENTIFIED (Over-identified)** assuming α₂ ≠ 0.

**Summary of Identification:**

| Equation | Order Condition | Rank Condition | Status |
|----------|-----------------|----------------|--------|
| Consumption | 2 > 1 (satisfied) | rank = 2 (satisfied if β₂ ≠ 0) | Over-identified |
| Investment | 2 > 1 (satisfied) | rank = 2 (satisfied if α₂ ≠ 0) | Over-identified |
| Identity | - | - | Not estimated (definitional) |

**(c) Appropriate Estimation Methods (3 marks)**

**For Consumption Function:**
- **Recommended Method: 2SLS (Two-Stage Least Squares)**
- Reason: The equation is over-identified, so 2SLS is appropriate
- Alternative: 3SLS if estimating the entire system and cross-equation correlations are important

**For Investment Function:**
- **Recommended Method: 2SLS (Two-Stage Least Squares)**
- Reason: The equation is over-identified
- Alternative: 3SLS for system estimation

**Why Not OLS?**
- OLS would be inconsistent because Y is endogenous
- Y is correlated with the error terms u₁ and u₂
- This simultaneity bias makes OLS inappropriate

**Why Not ILS (Indirect Least Squares)?**
- ILS is only applicable to exactly identified equations
- Both equations are over-identified
- ILS would give multiple solutions

**2SLS Procedure:**

For Consumption Function:
1. **Stage 1:** Regress Y on all exogenous variables (G, r, C₋₁)
   ```
   Ŷ = π̂₀ + π̂₁G + π̂₂r + π̂₃C₋₁
   ```

2. **Stage 2:** Regress C on Ŷ and C₋₁
   ```
   C = α₀ + α₁Ŷ + α₂C₋₁ + error
   ```

For Investment Function:
1. **Stage 1:** Same as above (Ŷ already obtained)

2. **Stage 2:** Regress I on Ŷ and r
   ```
   I = β₀ + β₁Ŷ + β₂r + error
   ```

**3SLS Consideration:**

If there is reason to believe that u₁ and u₂ are correlated across equations, 3SLS would be more efficient than 2SLS. However, 3SLS is more sensitive to specification errors.

From Durbin-Watson tables:
- dL ≈ 0.610
- dU ≈ 1.400

**Step 3: Decision**

| Region | Range | Decision |
|--------|-------|----------|
| Reject H₀ | d < dL = 0.610 | Positive autocorrelation |
| Inconclusive | 0.610 ≤ d ≤ 1.400 | Cannot conclude |

**Step 4: Check for Negative Autocorrelation**

**Do not reject H₀ for negative autocorrelation.**

**(d) Test Significance of β₁ (2 marks)**

**Step 1: Calculate Standard Error of β̂₁**

First, calculate σ̂²:
```
σ̂² = RSS/(n - k) = 1.334/(6 - 2) = 1.334/4 = 0.3335
```

Calculate Var(β̂₁):
```
Var(β̂₁) = σ̂² / Σ(X - X̄)²
        = σ̂² / [ΣX² - nX̄²]
        = 0.3335 / [91 - 6(3.5)²]
        = 0.3335 / 17.5
        = 0.01906

SE(β̂₁) = √0.01906 = 0.138
```

**Step 2: Calculate t-statistic**

```
H₀: β₁ = 0
H₁: β₁ ≠ 0

t = β̂₁ / SE(β̂₁)
  = 2.0 / 0.138
  = 14.49
```

**Step 3: Compare with Critical Value**

Degrees of freedom = n - k = 6 - 2 = 4

For two-tailed test at α = 0.05:
```
t₀.₀₂₅,₄ = 2.776
```

**Step 4: Decision**

Since |t| = 14.49 > t₀.₀₂₅,₄ = 2.776:

**Reject H₀ at 5% significance level.**

**Conclusion:** The slope coefficient β₁ is statistically significant at the 5% level. There is strong evidence that X has a significant effect on Y.

**Alternative: p-value approach**

With t = 14.49 and df = 4, the p-value is extremely small (p < 0.001), which is much less than α = 0.05. This confirms our rejection of H₀.

## 13.1 One-Page Summary: OLS Estimation

### Key Formulas

```
MODEL: Y = Xβ + u

OLS ESTIMATOR:
β̂ = (X'X)⁻¹X'Y

VARIANCE:
Var(β̂) = σ²(X'X)⁻¹

ESTIMATED VARIANCE:
σ̂² = RSS/(n-k) = e'e/(n-k)

PROPERTIES (under Gauss-Markov assumptions):
• Unbiased: E(β̂) = β
• Efficient: Minimum variance among linear unbiased estimators
• Consistent: plim(β̂) = β
• BLUE: Best Linear Unbiased Estimator
```

### Gauss-Markov Assumptions

1. **Linearity:** Y = Xβ + u
2. **Full Rank:** rank(X) = k
3. **Exogeneity:** E(u|X) = 0
4. **Homoscedasticity:** Var(uᵢ) = σ² for all i
5. **No Autocorrelation:** Cov(uᵢ, uⱼ) = 0 for i ≠ j

### Key Results

| Measure | Formula |
|---------|---------|
| TSS | Σ(Yᵢ - Ȳ)² |
| ESS | Σ(Ŷᵢ - Ȳ)² |
| RSS | Σeᵢ² = Σ(Yᵢ - Ŷᵢ)² |
| R² | ESS/TSS = 1 - RSS/TSS |
| Adjusted R² | 1 - (1-R²)(n-1)/(n-k) |
| F-statistic | [ESS/(k-1)] / [RSS/(n-k)] |
| t-statistic | β̂ⱼ / SE(β̂ⱼ) |

---

## 13.2 One-Page Summary: Heteroscedasticity

### Definition
```
Var(uᵢ) = σᵢ² ≠ σ² (variance not constant)
```

### Consequences on OLS

| Property | Effect |
|----------|--------|
| Unbiasedness | ✓ Retained |
| Consistency | ✓ Retained |
| Efficiency | ✗ Lost (not BLUE) |
| Variance estimates | ✗ Biased |
| t and F tests | ✗ Invalid |

### Detection Methods

| Test | Procedure |
|------|-----------|
| **Goldfeld-Quandt** | Split sample, compare RSS, F-test |
| **Breusch-Pagan** | Regress e²/σ̂² on Z, LM = ESS/2 ~ χ²(p) |
| **White's Test** | Regress e² on X, X², cross-products, nR² ~ χ²(q) |
| **Park Test** | Regress ln(e²) on ln(X) |
| **Glejser Test** | Regress |e| on functions of X |

### Remedies

1. **Weighted Least Squares (WLS):** Transform by 1/σᵢ
2. **Feasible GLS:** Estimate σᵢ², then apply WLS
3. **Robust Standard Errors:** White's heteroscedasticity-consistent SEs
4. **Transformation:** Log transformation often helps

### GLS Estimator
```
β̂ᵍˡˢ = (X'Ω⁻¹X)⁻¹X'Ω⁻¹Y

Where Ω = diag(σ₁², σ₂², ..., σₙ²)

Var(β̂ᵍˡˢ) = (X'Ω⁻¹X)⁻¹
```

---

## 13.3 One-Page Summary: Autocorrelation

### Definition
```
Cov(uₜ, uₛ) ≠ 0 for t ≠ s

AR(1) Process: uₜ = ρuₜ₋₁ + εₜ, |ρ| < 1
```

### Consequences on OLS

| Property | Effect |
|----------|--------|
| Unbiasedness | ✓ Retained |
| Consistency | ✓ Retained |
| Efficiency | ✗ Lost |
| Variance estimates | ✗ Underestimated (for ρ > 0) |
| t and F tests | ✗ Inflated (too many rejections) |
| R² | ✗ Inflated |

### Durbin-Watson Test

```
d = Σ(eₜ - eₜ₋₁)² / Σeₜ²

Relationship: d ≈ 2(1 - ρ̂)

Range: 0 ≤ d ≤ 4
• d = 0: Perfect positive autocorrelation
• d = 2: No autocorrelation
• d = 4: Perfect negative autocorrelation
```

### Decision Rules (Testing ρ > 0)

| Condition | Decision |
|-----------|----------|
| d < dL | Positive autocorrelation |
| d > dU | No positive autocorrelation |
| dL ≤ d ≤ dU | Inconclusive |

### Remedies

| Method | Description |
|--------|-------------|
| **Cochrane-Orcutt** | Iterative estimation of ρ |
| **Prais-Winsten** | Retains first observation |
| **Hildreth-Lu** | Grid search for ρ |
| **First Differences** | Yₜ - Yₜ₋₁ on Xₜ - Xₜ₋₁ |
| **GLS** | Transform using estimated ρ |

### GLS Transformation (AR(1))
```
Y*ₜ = Yₜ - ρYₜ₋₁
X*ₜ = Xₜ - ρXₜ₋₁

Then apply OLS to transformed model.
```

## 13.4 One-Page Summary: Multicollinearity

### Definition
```
High correlation among explanatory variables
Perfect: c₁X₁ + c₂X₂ + ... + cₖXₖ = 0
Imperfect: Near-linear relationships
```

### Consequences

| Effect | Description |
|--------|-------------|
| OLS still BLUE | ✓ Properties retained |
| Large variances | Var(β̂ⱼ) inflated |
| Wide confidence intervals | Less precise estimates |
| Insignificant t-ratios | Individual coefficients appear insignificant |
| High R² | Model fits well overall |
| Sensitive estimates | Small data changes cause large coefficient changes |
| Wrong signs | Coefficients may have unexpected signs |

### Detection Methods

| Method | Criterion |
|--------|-----------|
| **Correlation Matrix** | |rᵢⱼ| > 0.8 or 0.9 |
| **VIF** | VIFⱼ = 1/(1-Rⱼ²) > 10 indicates problem |
| **Tolerance** | TOLⱼ = 1/VIFⱼ < 0.1 indicates problem |
| **Condition Number** | κ > 30 indicates severe problem |
| **Eigenvalues** | Small eigenvalues indicate problem |

### Variance Inflation Factor
```
Var(β̂ⱼ) = σ² / [(1-Rⱼ²) × Σ(Xⱼ - X̄ⱼ)²]
         = σ² × VIFⱼ / Σ(Xⱼ - X̄ⱼ)²

VIFⱼ = 1/(1-Rⱼ²)

Where Rⱼ² = R² from regressing Xⱼ on all other X variables
```

### Remedies

| Method | Description |
|--------|-------------|
| **Do nothing** | If prediction is goal and multicollinearity persists |
| **Drop variables** | Remove redundant variables (may cause bias) |
| **Combine variables** | Create indices or ratios |
| **Ridge Regression** | Biased but lower MSE |
| **PCA Regression** | Use principal components |
| **More data** | Increase sample size |

### Ridge Regression
```
β̂ᵣᵢdgₑ = (X'X + kI)⁻¹X'Y

• Biased but lower variance
• MSE can be lower than OLS for appropriate k
• k = 0 gives OLS
```

## 13.5 One-Page Summary: Distributed Lag Models

### Infinite Distributed Lag Model
```
Yₜ = α + β₀Xₜ + β₁Xₜ₋₁ + β₂Xₜ₋₂ + ... + uₜ
   = α + Σᵢ₌₀^∞ βᵢXₜ₋ᵢ + uₜ
```

### Koyck Model

**Assumption:** βᵢ = β₀λⁱ, where 0 < λ < 1

**Transformed Model:**
```
Yₜ = α(1-λ) + β₀Xₜ + λYₜ₋₁ + vₜ

Where vₜ = uₜ - λuₜ₋₁ (MA(1) error)
```

### Interpretation

| Parameter | Meaning | Formula |
|-----------|---------|---------|
| β₀ | Short-run multiplier | Impact effect |
| λ | Decay rate | Speed of adjustment |
| β₀/(1-λ) | Long-run multiplier | Total cumulative effect |
| λ/(1-λ) | Mean lag | Average delay |
| -ln(2)/ln(λ) | Median lag | Half-life of effect |

### Problems in Estimation

| Problem | Description |
|---------|-------------|
| **Stochastic regressor** | Yₜ₋₁ is random |
| **Endogeneity** | Cov(Yₜ₋₁, vₜ) ≠ 0 |
| **Autocorrelated errors** | vₜ follows MA(1) |
| **D-W test invalid** | Biased toward 2 |
| **OLS inconsistent** | Due to endogeneity |

### Solutions

| Method | Description |
|--------|-------------|
| **Instrumental Variables** | Use Xₜ₋₁, Xₜ₋₂, Yₜ₋₂ as instruments |
| **Maximum Likelihood** | Full ML accounting for MA(1) errors |
| **Durbin's h-test** | h = ρ̂√(n/(1-n·Var(λ̂))) for autocorrelation |
| **Breusch-Godfrey** | LM test valid with lagged dependent variable |

## 13.6 One-Page Summary: Identification

### The Problem
```
Structural Form: BY + ΓX = U
Reduced Form: Y = ΠX + V, where Π = -B⁻¹Γ

Question: Can we recover B and Γ from Π?
```

### Types of Identification

| Type | Condition | Implication |
|------|-----------|-------------|
| **Under-identified** | K - k < g - 1 | Cannot estimate |
| **Exactly identified** | K - k = g - 1 | Unique solution |
| **Over-identified** | K - k > g - 1 | Multiple solutions, use 2SLS |

### Order Condition (Necessary)
```
K - kᵢ ≥ gᵢ - 1

Where:
K = total exogenous variables in system
kᵢ = exogenous variables in equation i
gᵢ = endogenous variables in equation i
```

### Rank Condition (Necessary & Sufficient)
```
rank(Aᵢ) = G - 1

Where Aᵢ = matrix of coefficients of excluded variables 
           in other equations
```

### Steps for Checking Identification

1. **Count variables:**
   - G = number of endogenous variables
   - K = number of exogenous variables

2. **For each equation, count:**
   - gᵢ = endogenous variables in equation
   - kᵢ = exogenous variables in equation

3. **Apply Order Condition:**
   - Check if K - kᵢ ≥ gᵢ - 1

4. **Apply Rank Condition:**
   - Form matrix Aᵢ of excluded variables' coefficients
   - Check if rank(Aᵢ) = G - 1

## 13.7 One-Page Summary: 2SLS Estimation

### Why OLS Fails in SEM
```
In structural equation: y = Zδ + u
Z contains endogenous variables
Cov(Z, u) ≠ 0
OLS is inconsistent
```

### 2SLS Procedure

**Stage 1: Purge Endogenous Variables**
```
Regress each endogenous RHS variable on ALL exogenous variables:
Ŷ = X(X'X)⁻¹X'Y = PₓY

Pₓ = X(X'X)⁻¹X' is the projection matrix
```

**Stage 2: Estimate Structural Equation**
```
Replace Y with Ŷ in structural equation:
y = Ẑδ + u*

Where Ẑ = [Ŷ, X₁] (instrumented endogenous + exogenous)
```

### 2SLS Estimator
```
δ̂₂ₛₗₛ = (Ẑ'Z)⁻¹Ẑ'y
       = (Z'PₓZ)⁻¹Z'Pₓy
```

### Properties

| Property | Status |
|----------|--------|
| Consistency | ✓ plim(δ̂₂ₛₗₛ) = δ |
| Asymptotic normality | ✓ √n(δ̂ - δ) → N(0, V) |
| Unbiasedness | ✗ Biased in finite samples |
| Efficiency | Less efficient than 3SLS |

### Variance
```
Var(δ̂₂ₛₗₛ) = σ²(Z'PₓZ)⁻¹

Estimated: V̂ar(δ̂₂ₛₗₛ) = σ̂²(Ẑ'Z)⁻¹

Where σ̂² = (y - Zδ̂)′(y - Zδ̂)/(n - k)
```

### Consistency Proof (Key Steps)
```
δ̂₂ₛₗₛ = δ + (Ẑ'Z)⁻¹Ẑ'u

plim(δ̂₂ₛₗₛ) = δ + plim(Ẑ'Z/n)⁻¹ · plim(Ẑ'u/n)
             = δ + Q⁻¹ · 0
             = δ

Because plim(Ẑ'u/n) = 0 (instruments uncorrelated with error)
```

| **2SLS** | Single eq. | ✓ | Moderate | Moderate | Low |
| **LIML** | Limited info | ✓ | Moderate+ | Moderate | Low |
| **3SLS** | System | ✓ | High | High | High |
| **FIML** | System | ✓ | Highest | Highest | Highest |

### When to Use Each

| Estimator | Use When |
|-----------|----------|
| **OLS** | Never for structural equations in SEM |
| **ILS** | Exactly identified equations only |
| **2SLS** | Over-identified equations, single equation focus, robustness needed |
| **LIML** | Over-identified, concerned about weak instruments |
| **3SLS** | All equations correctly specified, efficiency important |
| **FIML** | Correctly specified system, maximum efficiency needed |

### k-class Estimators
```
β̂ₖ = [Z'(I - kMₓ)Z]⁻¹Z'(I - kMₓ)y

Special cases:
• k = 0: OLS
• k = 1: 2SLS
• k = LIML value: LIML
```

### 3SLS Estimator
```
δ̂₃ₛₗₛ = [Z'(Σ̂⁻¹ ⊗ Pₓ)Z]⁻¹ Z'(Σ̂⁻¹ ⊗ Pₓ)y

Where Σ̂ = estimated error covariance matrix from 2SLS
```

### Key Relationships

1. **If Σ is diagonal:** 3SLS = 2SLS
2. **If all equations exactly identified:** 3SLS ≈ 2SLS asymptotically
3. **If system is recursive with diagonal Σ:** OLS is consistent

# CHAPTER 14: APPENDIX - FORMULA SHEET

## 14.1 Basic OLS Formulas

### Simple Regression (Y = β₀ + β₁X + u)

```
β̂₁ = Σ(Xᵢ - X̄)(Yᵢ - Ȳ) / Σ(Xᵢ - X̄)²
   = [ΣXY - nX̄Ȳ] / [ΣX² - nX̄²]
   = Sxy / Sxx

β̂₀ = Ȳ - β̂₁X̄

Var(β̂₁) = σ² / Σ(Xᵢ - X̄)²

Var(β̂₀) = σ²[1/n + X̄²/Σ(Xᵢ - X̄)²]

σ̂² = Σeᵢ² / (n - 2)
```

### Multiple Regression (Y = Xβ + u)

```
β̂ = (X'X)⁻¹X'Y

Var(β̂) = σ²(X'X)⁻¹

σ̂² = e'e / (n - k) = RSS / (n - k)

e = Y - Xβ̂ = MY

M = I - X(X'X)⁻¹X' (residual maker matrix)

P = X(X'X)⁻¹X' (projection matrix)
```

## 14.2 Goodness of Fit

```
TSS = Σ(Yᵢ - Ȳ)² = Y'MY (where M₀ = I - (1/n)11')

ESS = Σ(Ŷᵢ - Ȳ)² = β̂'X'M₀Xβ̂

RSS = Σeᵢ² = e'e = Y'MY

TSS = ESS + RSS

R² = ESS/TSS = 1 - RSS/TSS

Adjusted R² = 1 - [(1-R²)(n-1)/(n-k)]

F = [ESS/(k-1)] / [RSS/(n-k)]
  = [R²/(k-1)] / [(1-R²)/(n-k)]
```

## 14.3 Hypothesis Testing

```
t-statistic: t = (β̂ⱼ - βⱼ₀) / SE(β̂ⱼ) ~ t(n-k)

F-statistic: F = [(RSSᵣ - RSSᵤ)/q] / [RSSᵤ/(n-k)] ~ F(q, n-k)

Confidence Interval: β̂ⱼ ± tα/2,n-k × SE(β̂ⱼ)
```

## 14.4 GLS Formulas

```
Model: Y = Xβ + u, where Var(u) = σ²Ω

GLS Estimator:
β̂ᵍˡˢ = (X'Ω⁻¹X)⁻¹X'Ω⁻¹Y

Var(β̂ᵍˡˢ) = σ²(X'Ω⁻¹X)⁻¹

For heteroscedasticity (Ω = diag(σ₁², ..., σₙ²)):
WLS: β̂ʷˡˢ = (X'WX)⁻¹X'WY
Where W = diag(1/σ₁², ..., 1/σₙ²)

For AR(1) autocorrelation:
Ω has (i,j) element = ρ|i-j|
```

## 14.5 Diagnostic Tests

### Durbin-Watson Test
```
d = Σₜ₌₂ⁿ(eₜ - eₜ₋₁)² / Σₜ₌₁ⁿeₜ²

d ≈ 2(1 - ρ̂)

ρ̂ ≈ 1 - d/2

Range: 0 ≤ d ≤ 4
```

### Durbin's h-test (with lagged dependent variable)
```
h = ρ̂ √[n / (1 - n·Var(λ̂))]

Under H₀: h ~ N(0,1) asymptotically
```

### Breusch-Pagan Test
```
LM = (1/2) × ESS from auxiliary regression
Under H₀: LM ~ χ²(p)
```

### White's Test
```
W = nR² from auxiliary regression of e² on X, X², cross-products
Under H₀: W ~ χ²(q)
```

### Goldfeld-Quandt Test
```
F = RSS₂/RSS₁ (larger variance in numerator)
Under H₀: F ~ F(df₂, df₁)
```

### VIF (Variance Inflation Factor)
```
VIFⱼ = 1 / (1 - Rⱼ²)

Where Rⱼ² = R² from regressing Xⱼ on all other X variables
```

## 14.6 Ridge Regression

```
β̂ᵣᵢdgₑ = (X'X + kI)⁻¹X'Y

Bias: E(β̂ᵣᵢdgₑ) - β = -k(X'X + kI)⁻¹β

Var(β̂ᵣᵢdgₑ) = σ²(X'X + kI)⁻¹X'X(X'X + kI)⁻¹

MSE(β̂ᵣᵢdgₑ) = Var + Bias²
```

## 14.7 Distributed Lag Models

### Koyck Model
```
Original: Yₜ = α + Σᵢ₌₀^∞ β₀λⁱXₜ₋ᵢ + uₜ

Transformed: Yₜ = α(1-λ) + β₀Xₜ + λYₜ₋₁ + vₜ

Where vₜ = uₜ - λuₜ₋₁

Short-run multiplier: β₀
Long-run multiplier: β₀/(1-λ)
Mean lag: λ/(1-λ)
Median lag: -ln(2)/ln(λ)
```

### Almon Lag (Polynomial Distributed Lag)
```
βᵢ = a₀ + a₁i + a₂i² + ... + aₘiᵐ

Degree of polynomial: m
Maximum lag length: q
```

## 14.8 Simultaneous Equations

### Identification Conditions
```
Order Condition (Necessary):
K - kᵢ ≥ gᵢ - 1

Rank Condition (Necessary & Sufficient):
rank(Aᵢ) = G - 1

Where:
G = total endogenous variables
K = total exogenous variables
gᵢ = endogenous variables in equation i
kᵢ = exogenous variables in equation i
Aᵢ = matrix of excluded variables' coefficients in other equations
```

### 2SLS Estimator
```
Stage 1: Ŷ = PₓY = X(X'X)⁻¹X'Y

Stage 2: δ̂₂ₛₗₛ = (Ẑ'Z)⁻¹Ẑ'y

### 2SLS Estimator (continued)

```
Alternative form:
δ̂₂ₛₗₛ = (Z'PₓZ)⁻¹Z'Pₓy

Where:
Pₓ = X(X'X)⁻¹X' (projection matrix onto column space of X)
Z = [Y₁, X₁] (endogenous and exogenous RHS variables)
Ẑ = [Ŷ₁, X₁] (instrumented endogenous and exogenous variables)

Variance:
Var(δ̂₂ₛₗₛ) = σ²(Z'PₓZ)⁻¹

Estimated Variance:
V̂ar(δ̂₂ₛₗₛ) = σ̂²(Ẑ'Z)⁻¹

Where:
σ̂² = (y - Zδ̂₂ₛₗₛ)'(y - Zδ̂₂ₛₗₛ)/(n - k)
```

### 3SLS Estimator

```
System notation:
y = Zδ + u (stacked system)

Where:
y = [y₁, y₂, ..., yG]' (stacked dependent variables)
Z = diag[Z₁, Z₂, ..., ZG] (block diagonal matrix)
δ = [δ₁, δ₂, ..., δG]' (stacked parameters)
u = [u₁, u₂, ..., uG]' (stacked errors)

Error covariance:
E(uu') = Σ ⊗ Iₙ

Where Σ = [σᵢⱼ] is G×G covariance matrix of structural errors

3SLS Estimator:
δ̂₃ₛₗₛ = [Z'(Σ̂⁻¹ ⊗ Pₓ)Z]⁻¹ Z'(Σ̂⁻¹ ⊗ Pₓ)y

Where:
Σ̂ = estimated covariance matrix from 2SLS residuals
σ̂ᵢⱼ = êᵢ'êⱼ/n
⊗ = Kronecker product

Variance:
Var(δ̂₃ₛₗₛ) = [Z'(Σ⁻¹ ⊗ Pₓ)Z]⁻¹
```

### ILS (Indirect Least Squares)

```
For exactly identified equations:

Step 1: Estimate reduced form by OLS
Π̂ = (X'X)⁻¹X'Y

Step 2: Solve for structural parameters from:
Π = -B⁻¹Γ

Applicable only when K - kᵢ = gᵢ - 1 (exactly identified)
```

### k-class Estimators

```
General form:
β̂ₖ = [Z'(I - kMₓ)Z]⁻¹Z'(I - kMₓ)y

Where:
Mₓ = I - Pₓ = I - X(X'X)⁻¹X'

Special cases:
k = 0: OLS estimator
k = 1: 2SLS estimator
k = λ (LIML): LIML estimator

LIML k-value:
λ = smallest root of |W₁ - λW| = 0

Where:
W₁ = Y₁'MₓY₁ (matrix for included endogenous)
W = Y₁'M₁Y₁ (matrix for equation-specific residuals)
M₁ = I - X₁(X₁'X₁)⁻¹X₁'
```

### FIML (Full Information Maximum Likelihood)

```
Log-likelihood function:
ln L = constant - (n/2)ln|Σ| - (1/2)Σₜ uₜ'Σ⁻¹uₜ + n·ln|B|

Where:
uₜ = Byₜ + Γxₜ (structural errors)
B = matrix of endogenous variable coefficients
Γ = matrix of exogenous variable coefficients

Maximize ln L with respect to B, Γ, and Σ simultaneously.

Properties:
• Asymptotically efficient
• Uses all restrictions
• Sensitive to misspecification
• Computationally intensive
```

## 14.9 Matrix Algebra Essentials

### Key Matrix Operations

```
Transpose: (AB)' = B'A'

Inverse: (AB)⁻¹ = B⁻¹A⁻¹

Trace: tr(AB) = tr(BA)

Determinant: |AB| = |A||B|

Derivative: ∂(a'x)/∂x = a
           ∂(x'Ax)/∂x = 2Ax (if A symmetric)

Expected value: E(AXB) = AE(X)B

Variance: Var(AX) = A·Var(X)·A'
```

### Projection Matrix Properties

```
Pₓ = X(X'X)⁻¹X'

Properties:
• Pₓ² = Pₓ (idempotent)
• Pₓ' = Pₓ (symmetric)
• PₓX = X
• rank(Pₓ) = tr(Pₓ) = k

Residual maker:
Mₓ = I - Pₓ

Properties:
• Mₓ² = Mₓ (idempotent)
• Mₓ' = Mₓ (symmetric)
• MₓX = 0
• rank(Mₓ) = tr(Mₓ) = n - k
```

### Kronecker Product

```
Definition: A ⊗ B = [aᵢⱼB]

Properties:
• (A ⊗ B)(C ⊗ D) = AC ⊗ BD
• (A ⊗ B)⁻¹ = A⁻¹ ⊗ B⁻¹
• (A ⊗ B)' = A' ⊗ B'
• tr(A ⊗ B) = tr(A)·tr(B)
```

## 14.10 Statistical Distributions

### Key Distributions for Econometrics

```
Normal: X ~ N(μ, σ²)
• E(X) = μ
• Var(X) = σ²

Chi-square: X ~ χ²(n)
• E(X) = n
• Var(X) = 2n

t-distribution: t = Z/√(χ²/n) ~ t(n)
• E(t) = 0 (for n > 1)
• Var(t) = n/(n-2) (for n > 2)

F-distribution: F = (χ₁²/m)/(χ₂²/n) ~ F(m,n)
• E(F) = n/(n-2) (for n > 2)
```

### Asymptotic Results

```
Consistency: plim(θ̂ₙ) = θ

Asymptotic Normality: √n(θ̂ₙ - θ) →ᵈ N(0, V)

Slutsky's Theorem: If plim(Xₙ) = c and Yₙ →ᵈ Y, then:
• Xₙ + Yₙ →ᵈ c + Y
• XₙYₙ →ᵈ cY

Delta Method: If √n(θ̂ - θ) →ᵈ N(0, V), then:
√n(g(θ̂) - g(θ)) →ᵈ N(0, g'(θ)Vg'(θ)')
```
