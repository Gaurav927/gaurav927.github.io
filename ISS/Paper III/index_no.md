#### Key Formulas to Master

**Price Relatives:**
```
P₀₁ = (P₁/P₀) × 100

Laspeyre's Price Index:
L₀₁ = (Σp₁q₀/Σp₀q₀) × 100

Paasche's Price Index:
P₀₁ = (Σp₁q₁/Σp₀q₁) × 100

Fisher's Ideal Index:
F₀₁ = √(L₀₁ × P₀₁)

Marshall-Edgeworth Index:
ME₀₁ = [Σp₁(q₀+q₁)/Σp₀(q₀+q₁)] × 100
```

#### Tests for Index Numbers (VERY IMPORTANT for ISS)

| Test | Condition | Which Index Satisfies? |
|------|-----------|----------------------|
| **Time Reversal Test** | P₀₁ × P₁₀ = 1 | Fisher's ✓, Laspeyre's ✗, Paasche's ✗ |
| **Factor Reversal Test** | P₀₁ × Q₀₁ = (Σp₁q₁/Σp₀q₀) | Fisher's ✓ |
| **Circular Test** | P₀₁ × P₁₂ × P₂₀ = 1 | None of the standard indices |
| **Unit Test** | Independent of units | All standard indices ✓ |

**Best Source for Proofs:** S.C. Gupta & V.K. Kapoor - Contains complete proofs for all tests

### 📊 TOPIC C: Income Distribution

#### Pareto Distribution

**Best Sources:**
1. **S.C. Gupta & Kapoor** - Mathematical treatment
2. **Fundamentals of Mathematical Statistics** - S.C. Gupta (for distribution theory)

**Key Concepts:**
```
Pareto Distribution:
f(x) = (α × x_m^α) / x^(α+1),  x ≥ x_m

Pareto's Law of Income:
N = A/x^α
where N = number of persons with income ≥ x
      A, α = constants

Log form: log N = log A - α log x
```

#### Engel Curves

**Study Sources:**
- **Microeconomics** - H.L. Ahuja (Indian author, good coverage)
- **Economic Statistics** - V.K. Kapoor
- **Intermediate Microeconomics** - Varian (for advanced understanding)

**Key Points:**
```
Engel Curve: Relationship between income and quantity demanded
            of a commodity

Types:
├── Necessities: Curve flattens at higher income
├── Luxuries: Curve steepens at higher income
├── Inferior goods: Curve bends backward
└── Engel's Law: Food expenditure proportion decreases 
                 as income increases
```

#### Concentration Curve (Lorenz Curve)

**Best Source:** S.C. Gupta & Kapoor (complete treatment with Gini coefficient)

```
Lorenz Curve:
- X-axis: Cumulative % of population
- Y-axis: Cumulative % of income
- Line of equality: 45° line
- Gini Coefficient = Area between Lorenz curve and equality line
                     ÷ Total area under equality line
```

#### Must-Know Formulas & Derivations

```
LINK RELATIVE:
Lₜ = (Current Period Value / Previous Period Value) × 100

CHAIN INDEX FROM LINK RELATIVES:
Chain Index (Year n) = (L₁ × L₂ × ... × Lₙ) / 100^(n-1)

FIXED BASE FROM CHAIN BASE:
FBI(Year n) = (CBI₁ × CBI₂ × ... × CBIₙ) / 100^(n-1)

CHAIN BASE FROM FIXED BASE:
CBI(Year n) = [FBI(Year n) / FBI(Year n-1)] × 100

SPLICING FORMULA:
New Index = Old Index × (New Series Value at Overlap / Old Series Value at Overlap)

BASE SHIFTING:
New Index = (Old Index / Old Index at New Base Year) × 100
```

#### Complete Formula Reference

```
PRICE INDEX NUMBERS:

1. LASPEYRE'S PRICE INDEX (Base Year Quantity Weights):
   L₀₁ = (Σp₁q₀ / Σp₀q₀) × 100

2. PAASCHE'S PRICE INDEX (Current Year Quantity Weights):
   P₀₁ = (Σp₁q₁ / Σp₀q₁) × 100

3. FISHER'S IDEAL INDEX (Geometric Mean):
   F₀₁ = √(L₀₁ × P₀₁) = √[(Σp₁q₀/Σp₀q₀) × (Σp₁q₁/Σp₀q₁)] × 100

4. MARSHALL-EDGEWORTH INDEX (Average Quantity Weights):
   ME₀₁ = [Σp₁(q₀+q₁) / Σp₀(q₀+q₁)] × 100

5. DROBISCH INDEX (Arithmetic Mean):
   D₀₁ = (L₀₁ + P₀₁) / 2

6. BOWLEY'S INDEX (Arithmetic Mean):
   B₀₁ = (L₀₁ + P₀₁) / 2  [Same as Drobisch]

7. WALSH INDEX (Geometric Mean of Quantities):
   W₀₁ = [Σp₁√(q₀q₁) / Σp₀√(q₀q₁)] × 100


QUANTITY INDEX NUMBERS:

1. LASPEYRE'S QUANTITY INDEX:
   L_Q = (Σq₁p₀ / Σq₀p₀) × 100

2. PAASCHE'S QUANTITY INDEX:
   P_Q = (Σq₁p₁ / Σq₀p₁) × 100

3. FISHER'S QUANTITY INDEX:
   F_Q = √(L_Q × P_Q)
```

#### Bias in Index Numbers (Theory Question)

```
LASPEYRE'S UPWARD BIAS:
- Uses base year quantities (q₀)
- Doesn't account for substitution effect
- When prices rise, consumers substitute cheaper goods
- Laspeyre's overstates cost of living
- Formula assumes same basket purchased

PAASCHE'S DOWNWARD BIAS:
- Uses current year quantities (q₁)
- Reflects substitution already made
- Understates true cost increase
- Requires current quantity data (expensive)

MATHEMATICAL RELATIONSHIP:
Generally: Paasche ≤ Fisher ≤ Laspeyre
(When prices and quantities move inversely)

EQUALITY CONDITION:
L = P = F when: q₁/q₀ is same for all commodities
(i.e., proportional change in all quantities)
```

#### Complete Proofs (Must Memorize)

```
TEST 1: TIME REVERSAL TEST
═══════════════════════════
Condition: P₀₁ × P₁₀ = 1

FOR FISHER'S INDEX:
P₀₁ = √[(Σp₁q₀/Σp₀q₀) × (Σp₁q₁/Σp₀q₁)]
P₁₀ = √[(Σp₀q₁/Σp₁q₁) × (Σp₀q₀/Σp₁q₀)]

P₀₁ × P₁₀ = √[(Σp₁q₀/Σp₀q₀) × (Σp₁q₁/Σp₀q₁)] × √[(Σp₀q₁/Σp₁q₁) × (Σp₀q₀/Σp₁q₀)]
          = √[(Σp₁q₀/Σp₀q₀) × (Σp₀q₀/Σp₁q₀) × (Σp₁q₁/Σp₀q₁) × (Σp₀q₁/Σp₁q₁)]
          = √[1 × 1]
          = 1 ✓

FOR LASPEYRE'S INDEX:
L₀₁ = Σp₁q₀/Σp₀q₀
L₁₀ = Σp₀q₁/Σp₁q₁

L₀₁ × L₁₀ = (Σp₁q₀/Σp₀q₀) × (Σp₀q₁/Σp₁q₁) ≠ 1 in general ✗


TEST 2: FACTOR REVERSAL TEST
════════════════════════════
Condition: P₀₁ × Q₀₁ = V₀₁ = Σp₁q₁/Σp₀q₀

FOR FISHER'S INDEX:
P₀₁ = √[(Σp₁q₀/Σp₀q₀) × (Σp₁q₁/Σp₀q₁)]
Q₀₁ = √[(Σq₁p₀/Σq₀p₀) × (Σq₁p₁/Σq₀p₁)]

P₀₁ × Q₀₁ = √[(Σp₁q₀/Σp₀q₀) × (Σp₁q₁/Σp₀q₁) × (Σq₁p₀/Σq₀p₀) × (Σq₁p₁/Σq₀p₁)]

Note: Σp₁q₀ = Σq₀p₁, Σp₀q₀ = Σq₀p₀, etc.

= √[(Σp₁q₀/Σp₀q₀) × (Σp₁q₁/Σp₀q₁) × (Σp₀q₁/Σp₀q₀) × (Σp₁q₁/Σp₁q₀)]
= √[(Σp₁q₁/Σp₀q₀)²]
= Σp₁q₁/Σp₀q₀ = V₀₁ ✓


TEST 3: CIRCULAR TEST
═════════════════════
Condition: P₀₁ × P₁₂ × P₂₀ = 1

NO STANDARD WEIGHTED INDEX SATISFIES THIS TEST

Only Simple Geometric Mean of Price Relatives satisfies:
P₀₁ = ⁿ√(∏(p₁ᵢ/p₀ᵢ))

Proof that Fisher's fails:
The geometric mean structure doesn't cancel out 
when three periods are involved due to changing weights.


TEST 4: UNIT TEST (COMMENSURABILITY TEST)
═════════════════════════════════════════
Condition: Index should be independent of units of measurement

All standard indices (Laspeyre's, Paasche's, Fisher's) satisfy this test
because they use ratios of prices and quantities.
```

#### Complete Theory: Pareto Distribution

```
PARETO'S LAW OF INCOME DISTRIBUTION
═══════════════════════════════════

Basic Form:
N = A/x^α  or  N = A·x^(-α)

where:
N = Number of persons with income ≥ x
x = Income level
A = Constant (scale parameter)
α = Pareto constant/exponent (shape parameter)

Logarithmic Form:
log N = log A - α log x

This is linear in log-log scale:
Y = a + bX
where Y = log N, X = log x, a = log A, b = -α


PROBABILITY DISTRIBUTION FORM
═════════════════════════════

If X follows Pareto distribution with parameters (x_m, α):

CDF: F(x) = 1 - (x_m/x)^α,  x ≥ x_m

PDF: f(x) = (α·x_m^α)/x^(α+1),  x ≥ x_m

where x_m = minimum income (threshold)


MOMENTS OF PARETO DISTRIBUTION
══════════════════════════════

Mean: E(X) = α·x_m/(α-1),  exists only if α > 1

Variance: Var(X) = α·x_m²/[(α-1)²(α-2)],  exists only if α > 2

Mode: x_m (always at minimum value)

Median: x_m · 2^(1/α)


PARAMETER ESTIMATION
════════════════════

Method 1: Graphical Method
- Plot log N vs log x
- Fit straight line
- Slope = -α
- Intercept = log A

Method 2: Method of Least Squares
- Minimize Σ(log Nᵢ - log A + α log xᵢ)²
- Normal equations:
  Σlog N = n·log A - α·Σlog x
  Σ(log x)(log N) = log A·Σlog x - α·Σ(log x)²

Method 3: Maximum Likelihood
- α̂ = n / Σlog(xᵢ/x_m)
- x̂_m = min(x₁, x₂, ..., xₙ)


INTERPRETATION OF α
═══════════════════

α is inversely related to inequality:
- Higher α → Lower inequality (steeper curve)
- Lower α → Higher inequality (flatter curve)

Typical values: α ranges from 1.5 to 3 for most economies

Relationship with Gini Coefficient:
G = 1/(2α - 1),  for α > 0.5


LIMITATIONS OF PARETO'S LAW
═══════════════════════════

1. Applies only to higher income groups
   - Does not fit well for lower/middle incomes
   - Originally fitted to top 10-20% of population

2. Assumes continuous distribution
   - Income is often discrete
   - Grouped data issues

3. Constant α assumption
   - α may vary over time
   - Different for different economies

4. Does not explain income generation
   - Purely descriptive, not causal
   - No economic theory behind it

5. Sensitive to data quality
   - Tax evasion affects data
   - Under-reporting of high incomes
```

#### Numerical Example (Step-by-Step)

```
EXAMPLE: Fitting Pareto's Law

Data:
| x (Income '000) | 5    | 10   | 15   | 20   | 25   |
| N (Persons)     | 1000 | 400  | 200  | 120  | 80   |

Step 1: Take logarithms
| log x  | 0.699 | 1.000 | 1.176 | 1.301 | 1.398 |
| log N  | 3.000 | 2.602 | 2.301 | 2.079 | 1.903 |

Step 2: Calculate required sums
Σlog x = 5.574
Σlog N = 11.885
Σ(log x)² = 6.458
Σ(log x)(log N) = 12.684
n = 5

Step 3: Apply normal equations
11.885 = 5·log A - α·5.574  ... (i)
12.684 = log A·5.574 - α·6.458  ... (ii)

Step 4: Solve simultaneously
From (i): log A = (11.885 + 5.574α)/5 = 2.377 + 1.115α

Substitute in (ii):
12.684 = (2.377 + 1.115α)·5.574 - 6.458α
12.684 = 13.245 + 6.215α - 6.458α
12.684 = 13.245 - 0.243α
α = (13.245 - 12.684)/0.243 = 2.309

log A = 2.377 + 1.115(2.309) = 4.951
A = 10^4.951 = 89,337

Step 5: Fitted equation
N = 89,337/x^2.309

Step 6: Verify/Predict
For x = 12 (₹12,000):
N = 89,337/(12)^2.309 = 89,337/287.8 ≈ 310 persons
```

#### Complete Theory: Engel Curves

```
ENGEL'S LAW
═══════════

Statement: "The proportion of income spent on food decreases 
as income increases, other things remaining constant."

Mathematical Form:
Let F = Food expenditure, Y = Income
Then: d(F/Y)/dY < 0

Implications:
1. Food is a necessity
2. Income elasticity of food < 1
3. As countries develop, food share in budget decreases


ENGEL CURVE
═══════════

Definition: A graph showing the relationship between 
consumer's income and quantity demanded of a good,
holding prices constant.

Equation: Q = f(Y), where Q = quantity, Y = income

Graphical Representation:

    Q                          Q                          Q
    │    ╱                     │         ╱                │    ╲
    │   ╱                      │        ╱                 │     ╲
    │  ╱                       │       ╱                  │      ╲
    │ ╱                        │     ╱╱                   │       ╲
    │╱                         │   ╱╱                     │        ╲
    └──────────── Y            └──────────── Y            └──────────── Y
    NECESSITY                  LUXURY                     INFERIOR GOOD
    (Concave)                  (Convex)                   (Backward bending)


TYPES OF GOODS BASED ON ENGEL CURVES
════════════════════════════════════

1. NECESSITIES:
   - Engel curve: Concave (flattens out)
   - Income elasticity: 0 < η < 1
   - Examples: Food, basic clothing, utilities
   - As income rises, quantity increases but at decreasing rate

2. LUXURIES:
   - Engel curve: Convex (steepens)
   - Income elasticity: η > 1
   - Examples: Jewelry, vacations, premium goods
   - As income rises, quantity increases at increasing rate

3. INFERIOR GOODS:
   - Engel curve: Backward bending (after a point)
   - Income elasticity: η < 0
   - Examples: Coarse grains, public transport
   - As income rises beyond a point, quantity decreases


ENGEL COEFFICIENT
═════════════════

Definition: Proportion of income spent on food

E = Food Expenditure / Total Income = F/Y

Properties:
- 0 < E < 1
- Higher E indicates lower standard of living
- Used as indicator of economic development

International Comparison:
- Developed countries: E ≈ 10-15%
- Developing countries: E ≈ 30-50%
- Least developed: E > 50%


FUNCTIONAL FORMS FOR ENGEL CURVES
═════════════════════════════════

1. Linear Form:
   Q = a + bY
   Income elasticity: η = bY/Q = bY/(a+bY)

2. Log-Linear (Constant Elasticity):
   log Q = a + b·log Y
   Income elasticity: η = b (constant)

3. Semi-Log Form:
   Q = a + b·log Y
   Income elasticity: η = b/Q

4. Working-Leser Form:
   w = a + b·log Y
   where w = budget share = pQ/Y

5. Quadratic Form:
   Q = a + bY + cY²
   Allows for non-monotonic relationship


INCOME ELASTICITY FROM ENGEL CURVE
══════════════════════════════════

Definition: η = (dQ/dY) × (Y/Q) = (% change in Q)/(% change in Y)

For Linear Engel Curve Q = a + bY:
η = b × Y/(a + bY)

At mean values:
η = b × Ȳ/(a + bȲ) = b × Ȳ/Q̄

Classification:
- η > 1: Luxury good
- 0 < η < 1: Necessity
- η < 0: Inferior good
- η = 1: Unitary elastic (homothetic preferences)
```

#### Complete Theory: Lorenz Curve & Gini Coefficient

```
LORENZ CURVE
════════════

Definition: A graphical representation of income/wealth distribution
showing the cumulative percentage of total income received by
cumulative percentage of population (ranked from poorest to richest).

Construction:
- X-axis: Cumulative % of population (from lowest to highest income)
- Y-axis: Cumulative % of income

Line of Perfect Equality: 45° diagonal line
- Represents equal distribution
- Every x% of population receives x% of income

Lorenz Curve Properties:
1. Always lies below (or on) the line of equality
2. Starts at origin (0,0)
3. Ends at (100,100)
4. Convex to the x-axis
5. Slope increases as we move along the curve

Interpretation:
- Closer to diagonal → More equal distribution
- Further from diagonal → More unequal distribution
- Curves can intersect (different inequality patterns)


GRAPHICAL REPRESENTATION
════════════════════════

Cumulative % of Income
    100│                    ╱
       │                  ╱╱
       │                ╱╱
       │              ╱╱    ← Line of Equality
       │            ╱╱
       │          ╱╱
       │        ╱╱╱╱╱╱╱╱ ← Lorenz Curve
       │      ╱╱╱╱
       │    ╱╱╱
       │  ╱╱
       │╱╱
     0 └────────────────────
       0                  100
         Cumulative % of Population

Area A = Area between diagonal and Lorenz Curve
Area B = Area below Lorenz Curve
Total Area below diagonal = A + B = 0.5 (for 0-1 scale)


GINI COEFFICIENT
════════════════

Definition: G = A/(A+B) = A/0.5 = 2A

Alternative: G = 1 - 2B (where B = area under Lorenz Curve)

Range: 0 ≤ G ≤ 1

Interpretation:
- G = 0: Perfect equality (everyone has same income)
- G = 1: Perfect inequality (one person has all income)
- Higher G: Greater inequality


CALCULATION METHODS
═══════════════════

Method 1: Trapezoidal Approximation

G = 1 - Σᵢ(Xᵢ - Xᵢ₋₁)(Yᵢ + Yᵢ₋₁)

where:
Xᵢ = Cumulative proportion of population up to group i
Yᵢ = Cumulative proportion of income up to group i


Method 2: Brown's Formula

G = 1 - Σᵢ(Xᵢ - Xᵢ₋₁)(Yᵢ + Yᵢ₋₁)

Same as above, commonly used for grouped data.


Method 3: Covariance Formula (for individual data)

G = (2/n²μ) Σᵢ iYᵢ - (n+1)/n

where observations are arranged in ascending order


Method 4: Using Pareto's α

G = 1/(2α - 1), for Pareto distribution with α > 0.5
```

#### Three Methods of Measurement

```
METHOD 1: PRODUCT METHOD (VALUE ADDED METHOD)
═════════════════════════════════════════════

Concept: Sum of value added by all producing units in the economy

Formula:
GDP_FC = Σ (Gross Value Added by all sectors)
       = Σ (Value of Output - Value of Intermediate Consumption)

Sector-wise:
GDP = VA_Agriculture + VA_Industry + VA_Services

Value Added = Gross Output - Intermediate Inputs
            = Sales + Change in Stock - Purchases from other firms

Example:
┌─────────────────────────────────────────────────────────────┐
│ Stage        │ Value of Output │ Intermediate │ Value Added │
├─────────────────────────────────────────────────────────────┤
│ Farmer       │ ₹100 (wheat)    │ ₹0           │ ₹100        │
│ Miller       │ ₹150 (flour)    │ ₹100         │ ₹50         │
│ Baker        │ ₹200 (bread)    │ ₹150         │ ₹50         │
│ Retailer     │ ₹250 (final)    │ ₹200         │ ₹50         │
├─────────────────────────────────────────────────────────────┤
│ Total        │ ₹700            │ ₹450         │ ₹250        │
└─────────────────────────────────────────────────────────────┘

Note: Final Value (₹250) = Sum of Value Added (₹250)
      This avoids double counting

Advantages:
- Shows contribution of each sector
- Reveals structural changes in economy
- Useful for planning

Difficulties:
- Identifying all producing units
- Valuing non-marketed output (subsistence farming)
- Distinguishing intermediate from final goods
- Imputing value of owner-occupied housing


METHOD 2: INCOME METHOD (FACTOR PAYMENT METHOD)
═══════════════════════════════════════════════

Concept: Sum of all factor incomes earned in production

Formula:
NI = W + R + I + P + MI

where:
W = Wages and Salaries (Compensation of Employees)
R = Rent (including imputed rent)
I = Interest (net)
P = Profits (Corporate + Non-corporate)
MI = Mixed Income of Self-employed

Detailed Breakdown:
┌─────────────────────────────────────────────────────────────┐
│ Component                              │ Includes           │
├─────────────────────────────────────────────────────────────┤
│ Compensation of Employees              │ Wages, salaries,   │
│                                        │ bonuses, PF, etc.  │
├─────────────────────────────────────────────────────────────┤
│ Rent                                   │ Actual rent +      │
│                                        │ Imputed rent       │
├─────────────────────────────────────────────────────────────┤
│ Interest                               │ Net interest       │
│                                        │ (received - paid)  │
├─────────────────────────────────────────────────────────────┤
│ Profit                                 │ Dividends +        │
│                                        │ Retained earnings +│
│                                        │ Corporate tax      │
├─────────────────────────────────────────────────────────────┤
│ Mixed Income                           │ Income of self-    │
│                                        │ employed (farmers, │
│                                        │ shopkeepers, etc.) │
└─────────────────────────────────────────────────────────────┘

Advantages:
- Shows functional distribution of income
- Useful for analyzing income inequality
- Helps in tax policy formulation

Difficulties:
- Separating factor incomes in unorganized sector
- Mixed income problem (self-employed)
- Transfer payments vs factor payments
- Imputing rent for owner-occupied houses


METHOD 3: EXPENDITURE METHOD (FINAL OUTPUT METHOD)
══════════════════════════════════════════════════

Concept: Sum of all final expenditures in the economy

Formula:
GDP_MP = C + I + G + (X - M)

Detailed Components:
┌─────────────────────────────────────────────────────────────┐
│ C = Private Final Consumption Expenditure (PFCE)           │
│     - Household spending on goods and services             │
│     - Includes imputed values (own consumption)            │
├─────────────────────────────────────────────────────────────┤
│ I = Gross Domestic Capital Formation (GDCF)                │
│     = Gross Fixed Capital Formation (GFCF)                 │
│       + Change in Stocks (Inventory Investment)            │
│     - Business investment in plant, equipment, buildings   │
├─────────────────────────────────────────────────────────────┤
│ G = Government Final Consumption Expenditure (GFCE)        │
│     - Government spending on goods and services            │
│     - Excludes transfer payments                           │
├─────────────────────────────────────────────────────────────┤
│ (X - M) = Net Exports                                      │
│     X = Exports of goods and services                      │
│     M = Imports of goods and services                      │
└─────────────────────────────────────────────────────────────┘

Advantages:
- Shows demand composition
- Useful for Keynesian analysis
- Helps in understanding business cycles

Difficulties:
- Distinguishing final from intermediate expenditure
- Valuing government services
- Inventory valuation
- Estimating household consumption
```

#### Complete Theory: Input-Output Analysis

```
INPUT-OUTPUT MODEL: BASIC FRAMEWORK
═══════════════════════════════════

Developed by: Wassily Leontief (Nobel Prize 1973)

Purpose: To analyze inter-industry relationships and 
         determine output levels needed to meet final demand

Basic Idea: Output of each industry is used as:
           1. Intermediate input by other industries
           2. Final demand (consumption, investment, exports)


STRUCTURE OF INPUT-OUTPUT TABLE
═══════════════════════════════

                    PURCHASING SECTORS
                    ┌─────────────────────────────────────┐
                    │ Ind.1 │ Ind.2 │...│ Ind.n │ Final  │ Total
                    │       │       │   │       │ Demand │ Output
    ┌───────────────┼───────┼───────┼───┼───────┼────────┼───────
    │ Industry 1    │ x₁₁   │ x₁₂   │...│ x₁ₙ   │ F₁     │ X₁
P   │ Industry 2    │ x₂₁   │ x₂₂   │...│ x₂ₙ   │ F₂     │ X₂
R   │ ...           │ ...   │ ...   │...│ ...   │ ...    │ ...
O   │ Industry n    │ xₙ₁   │ xₙ₂   │...│ xₙₙ   │ Fₙ     │ Xₙ
D   ├───────────────┼───────┼───────┼───┼───────┼────────┼───────
U   │ Primary       │ V₁    │ V₂    │...│ Vₙ    │        │
C   │ Inputs        │       │       │   │       │        │
I   ├───────────────┼───────┼───────┼───┼───────┼────────┼───────
N   │ Total Input   │ X₁    │ X₂    │...│ Xₙ    │        │
G   └───────────────┴───────┴───────┴───┴───────┴────────┴───────

where:
xᵢⱼ = Flow from industry i to industry j
Fᵢ = Final demand for output of industry i
Xᵢ = Total output of industry i
Vⱼ = Primary inputs (value added) in industry j


BALANCE EQUATIONS
═════════════════

Row Balance (for each industry i):
Xᵢ = Σⱼxᵢⱼ + Fᵢ
Total Output = Intermediate Sales + Final Demand

Column Balance (for each industry j):
Xⱼ = Σᵢxᵢⱼ + Vⱼ
Total Input = Intermediate Purchases + Primary Inputs


TECHNICAL COEFFICIENTS
══════════════════════

Definition: aᵢⱼ = xᵢⱼ/Xⱼ

Interpretation: Amount of output from industry i required 
               to produce one unit of output in industry j

Technical Coefficient Matrix:
A = [aᵢⱼ] = [xᵢⱼ/Xⱼ]

Properties:
- 0 ≤ aᵢⱼ ≤ 1
- Column sums < 1 (for viable economy)
- Assumed constant (fixed proportions)


LEONTIEF MODEL
══════════════

From balance equation:
Xᵢ = Σⱼaᵢⱼxⱼ + Fᵢ

In matrix form:
X = AX + F

Solving for X:
X - AX = F
(I - A)X = F
X = (I - A)⁻¹F

where:
X = Output vector (n × 1)
A = Technical coefficient matrix (n × n)
F = Final demand vector (n × 1)
(I - A)⁻¹ = Leontief Inverse Matrix


LEONTIEF INVERSE MATRIX
═══════════════════════

L = (I - A)⁻¹

Properties:
1. All elements are non-negative (if Hawkins-Simon conditions hold)
2. Diagonal elements ≥ 1
3. lᵢⱼ represents total (direct + indirect) requirement from 
   industry i per unit of final demand from industry j

Interpretation of lᵢⱼ:
- Direct requirement: aᵢⱼ
- First round indirect: Σₖaᵢₖaₖⱼ
- Total requirement: lᵢⱼ = aᵢⱼ + Σₖaᵢₖaₖⱼ + Σₖ,ₘaᵢₖaₖₘaₘⱼ + ...

Series Expansion:
(I - A)⁻¹ = I + A + A² + A³ + ... (if convergent)


HAWKINS-SIMON CONDITIONS
════════════════════════

For (I - A)⁻¹ to exist with all non-negative elements:

All principal minors of (I - A) must be positive:

For 2×2 case:
1. (1 - a₁₁) > 0
2. (1 - a₁₁)(1 - a₂₂) - a₁₂a₂₁ > 0
   i.e., |I - A| > 0

Economic Interpretation:
- Each industry must use less than one unit of its own 
  output to produce one unit of output
- The economy must be productive (not all output used up 
  in intermediate production)
```

#### Numerical Example: Complete Solution

```
NUMERICAL EXAMPLE: INPUT-OUTPUT ANALYSIS
════════════════════════════════════════

Given Transaction Matrix:
┌────────────┬────────────┬────────────┬──────────┬────────┐
│            │ Industry 1 │ Industry 2 │ Final D. │ Total  │
├────────────┼────────────┼────────────┼──────────┼────────┤
│ Industry 1 │ 150        │ 500        │ 350      │ 1000   │
│ Industry 2 │ 200        │ 100        │ 700      │ 1000   │
│ Value Added│ 650        │ 400        │          │        │
│ Total Input│ 1000       │ 1000       │          │        │
└────────────┴────────────┴────────────┴──────────┴────────┘

STEP 1: Calculate Technical Coefficients

a₁₁ = x₁₁/X₁ = 150/1000 = 0.15
a₁₂ = x₁₂/X₂ = 500/1000 = 0.50
a₂₁ = x₂₁/X₁ = 200/1000 = 0.20
a₂₂ = x₂₂/X₂ = 100/1000 = 0.10

Technical Coefficient Matrix:
A = │ 0.15  0.50 │
    │ 0.20  0.10 │

STEP 2: Calculate (I - A)

I - A = │ 1  0 │ - │ 0.15  0.50 │
        │ 0  1 │   │ 0.20  0.10 │

      = │ 0.85  -0.50 │
        │-0.20   0.90 │

STEP 3: Check Hawkins-Simon Conditions

1. (1 - a₁₁) = 0.85 > 0 ✓
2. (1 - a₂₂) = 0.90 > 0 ✓
3. |I - A| = (0.85)(0.90) - (-0.50)(-0.20)
           = 0.765 - 0.10
           = 0.665 > 0 ✓

All conditions satisfied!

For 2×2 matrix: │a  b│⁻¹ = (1/|A|)  │ d  -b│
                │c  d│              │-c   a│

          = │ 1.353   0.752 │
            │ 0.301   1.278 │

│X₁│   │ 1.353   0.752 │   │350│
│X₂│ = │ 0.301   1.278 │ × │700│

If new final demand F' = │200│
                         │300│
```

#### Assumptions and Limitations

```
ASSUMPTIONS OF INPUT-OUTPUT MODEL
═════════════════════════════════

1. FIXED TECHNICAL COEFFICIENTS (Constant Returns to Scale)
   - aᵢⱼ remains constant regardless of output level
   - No substitution between inputs
   - Linear production functions

2. HOMOGENEOUS PRODUCTS
   - Each industry produces single homogeneous output
   - No joint products
   - No quality differences

3. NO CAPACITY CONSTRAINTS
   - Industries can expand output without limit
   - No supply bottlenecks
   - Unlimited factor availability

4. FIXED PRICES
   - All transactions at constant prices
   - No inflation effects
   - Base year prices used

5. DEMAND DETERMINED OUTPUT
   - Supply adjusts to demand
   - No inventory adjustments
   - Instantaneous adjustment

6. CLOSED ECONOMY (Basic Model)
   - No imports/exports in basic model
   - Extended models include trade


LIMITATIONS
═══════════

1. RIGIDITY OF COEFFICIENTS
   - Technological change not captured
   - Substitution possibilities ignored
   - Economies of scale not reflected

2. AGGREGATION PROBLEMS
   - Industries are aggregates of many firms
   - Product heterogeneity within industries
   - Different technologies within industry

3. DATA REQUIREMENTS
   - Massive data collection needed
   - Time lag in compilation
   - Costly and time-consuming

4. STATIC NATURE
   - No time dimension
   - Cannot capture dynamic adjustments
   - Investment effects not modeled

5. LINEAR ASSUMPTIONS
   - Real production often non-linear
   - Threshold effects ignored
   - Indivisibilities not captured

6. NO PRICE MECHANISM
   - Prices assumed fixed
   - No market clearing mechanism
   - Resource allocation not optimal
```

#### Complete Theory: Demand Analysis

```
DEMAND FUNCTION
═══════════════

General Form:
Qd = f(P, Y, Ps, Pc, T, E, A, ...)

where:
Qd = Quantity demanded
P = Own price
Y = Consumer income
Ps = Price of substitutes
Pc = Price of complements
T = Consumer tastes/preferences
E = Consumer expectations
A = Advertising expenditure


COMMON FUNCTIONAL FORMS
═══════════════════════

1. LINEAR DEMAND FUNCTION:
   Q = a - bP + cY + dPs
   
   Properties:
   - Constant slope
   - Variable elasticity along curve
   - Easy to estimate

2. LOG-LINEAR (CONSTANT ELASTICITY):
   ln Q = a - b ln P + c ln Y + d ln Ps
   
   Or: Q = A × P^(-b) × Y^c × Ps^d
   
   Properties:
   - Constant elasticities (b, c, d)
   - Multiplicative form
   - Widely used in empirical work

3. SEMI-LOG FORM:
   Q = a - b ln P
   Or: ln Q = a - bP
   
4. QUADRATIC FORM:
   Q = a - bP + cP²


PROPERTIES OF DEMAND FUNCTIONS
══════════════════════════════

1. HOMOGENEITY OF DEGREE ZERO
   Q(λP, λPs, λPc, λY) = Q(P, Ps, Pc, Y)
   
   Interpretation: Doubling all prices and income 
   leaves quantity demanded unchanged
   (No money illusion)

2. ADDING-UP (ENGEL AGGREGATION)
   Σwᵢηᵢ = 1
   
   where wᵢ = budget share of good i
         ηᵢ = income elasticity of good i
   
   Interpretation: Weighted sum of income elasticities = 1

3. SYMMETRY (SLUTSKY SYMMETRY)
   ∂hᵢ/∂Pⱼ = ∂hⱼ/∂Pᵢ
   
   where h = Hicksian (compensated) demand

4. NEGATIVITY
   Own substitution effect is negative
   ∂hᵢ/∂Pᵢ < 0


ELASTICITY CONCEPTS
═══════════════════

1. PRICE ELASTICITY OF DEMAND (Ed):

   Point Elasticity:
   Ed = (∂Q/∂P) × (P/Q) = (dQ/dP) × (P/Q)
   
   Arc Elasticity:
   Ed = [(Q₂-Q₁)/(Q₂+Q₁)] / [(P₂-P₁)/(P₂+P₁)]
      = [(Q₂-Q₁)/(P₂-P₁)] × [(P₂+P₁)/(Q₂+Q₁)]

   Classification:
   |Ed| > 1: Elastic demand
   |Ed| = 1: Unitary elastic
   |Ed| < 1: Inelastic demand
   |Ed| = 0: Perfectly inelastic
   |Ed| = ∞: Perfectly elastic


2. INCOME ELASTICITY OF DEMAND (Ey):

   Ey = (∂Q/∂Y) × (Y/Q)

   Classification:
   Ey > 1: Luxury good
   0 < Ey < 1: Necessity
   Ey < 0: Inferior good
   Ey = 0: Income-independent


3. CROSS PRICE ELASTICITY (Exy):

   Exy = (∂Qx/∂Py) × (Py/Qx)

   Classification:
   Exy > 0: Substitutes
   Exy < 0: Complements
   Exy = 0: Independent goods


RELATIONSHIP: ELASTICITY AND TOTAL REVENUE
══════════════════════════════════════════

Total Revenue: TR = P × Q

Marginal Revenue: MR = dTR/dQ = P(1 - 1/|Ed|)

Relationship:
┌─────────────────┬──────────────┬─────────────────┐
│ If |Ed|         │ Price ↑      │ Price ↓         │
├─────────────────┼──────────────┼─────────────────┤
│ > 1 (Elastic)   │ TR ↓         │ TR ↑            │
│ = 1 (Unitary)   │ TR unchanged │ TR unchanged    │
│ < 1 (Inelastic) │ TR ↑         │ TR ↓            │
└─────────────────┴──────────────┴─────────────────┘


ELASTICITY FORMULAS FOR DIFFERENT DEMAND FUNCTIONS
══════════════════════════════════════════════════

1. Linear: Q = a - bP
   Ed = -b × (P/Q) = -bP/(a-bP)
   
   At P = 0: Ed = 0
   At Q = 0: Ed = -∞
   At midpoint: Ed = -1

2. Log-Linear: ln Q = a - b ln P
   Ed = -b (constant)

3. Rectangular Hyperbola: PQ = k
   Ed = -1 (always unitary)

4. Q = aP^(-b)
   Ed = -b (constant)
```

#### Numerical Examples

```
EXAMPLE 1: Calculating Elasticities from Demand Function
════════════════════════════════════════════════════════

Given: Q = 200 - 5P + 0.1Y + 2Ps

where P = 20, Y = 1000, Ps = 30

Step 1: Calculate Q
Q = 200 - 5(20) + 0.1(1000) + 2(30)
Q = 200 - 100 + 100 + 60 = 260

Step 2: Price Elasticity
Ed = (∂Q/∂P) × (P/Q)
∂Q/∂P = -5
Ed = -5 × (20/260) = -100/260 = -0.385

Interpretation: Inelastic demand (|Ed| < 1)
1% increase in price → 0.385% decrease in quantity

Step 3: Income Elasticity
Ey = (∂Q/∂Y) × (Y/Q)
∂Q/∂Y = 0.1
Ey = 0.1 × (1000/260) = 100/260 = 0.385

Interpretation: Necessity (0 < Ey < 1)
1% increase in income → 0.385% increase in quantity

Step 4: Cross Price Elasticity
Exy = (∂Q/∂Ps) × (Ps/Q)
∂Q/∂Ps = 2
Exy = 2 × (30/260) = 60/260 = 0.231

Interpretation: Substitute good (Exy > 0)
1% increase in substitute price → 0.231% increase in quantity


EXAMPLE 2: Estimating Demand Function using OLS
═══════════════════════════════════════════════

Data:
| P  | 5  | 10 | 15 | 20 | 25 |
| Q  | 80 | 65 | 50 | 35 | 20 |

Model: Q = a + bP

Step 1: Calculate means
P̄ = (5+10+15+20+25)/5 = 75/5 = 15
Q̄ = (80+65+50+35+20)/5 = 250/5 = 50

Step 2: Calculate deviations and products
| P  | Q  | P-P̄ | Q-Q̄ | (P-P̄)² | (P-P̄)(Q-Q̄) |
|----|----|----- |------|--------|-------------|
| 5  | 80 | -10  | 30   | 100    | -300        |
| 10 | 65 | -5   | 15   | 25     | -75         |
| 15 | 50 | 0    | 0    | 0      | 0           |
| 20 | 35 | 5    | -15  | 25     | -75         |
| 25 | 20 | 10   | -30  | 100    | -300        |
|Sum |    |      |      | 250    | -750        |

Step 3: Calculate slope (b)
b = Σ(P-P̄)(Q-Q̄) / Σ(P-P̄)²
b = -750/250 = -3

Step 4: Calculate intercept (a)
a = Q̄ - bP̄
a = 50 - (-3)(15) = 50 + 45 = 95

Step 5: Estimated Demand Function
Q = 95 - 3P

Step 6: Price Elasticity at Mean
Ed = b × (P̄/Q̄) = -3 × (15/50) = -0.9

Interpretation: Slightly inelastic at mean values


EXAMPLE 3: Arc Elasticity Calculation
═════════════════════════════════════

Given: Price changes from ₹10 to ₹12
       Quantity changes from 100 to 80 units

Arc Elasticity Formula:
Ed = [(Q₂-Q₁)/(Q₂+Q₁)] / [(P₂-P₁)/(P₂+P₁)]

Ed = [(80-100)/(80+100)] / [(12-10)/(12+10)]
   = [-20/180] / [2/22]
   = [-0.111] / [0.091]
   = -1.22

Interpretation: Elastic demand (|Ed| > 1)
```

### 5.7 Common Mistakes to Avoid

```
INDEX NUMBERS
═════════════

✗ MISTAKE: Confusing Laspeyre's and Paasche's weights
  Laspeyre's uses BASE year quantities (q₀)
  Paasche's uses CURRENT year quantities (q₁)

✗ MISTAKE: Forgetting to multiply by 100 in index formulas

✗ MISTAKE: Wrong formula for chain to fixed base conversion
  CORRECT: FBI_n = (CBI₁ × CBI₂ × ... × CBI_n)/100^(n-1)

✗ MISTAKE: Claiming Fisher's satisfies Circular Test
  Fisher's FAILS Circular Test (only GM of relatives passes)

✗ MISTAKE: Not verifying Time Reversal numerically
  Always check: P₀₁ × P₁₀ = 1 (or close to 1)


INCOME DISTRIBUTION
═══════════════════

✗ MISTAKE: Forgetting conditions for Pareto mean/variance
  Mean exists only if α > 1
  Variance exists only if α > 2

✗ MISTAKE: Wrong sign in Pareto fitting
  log N = log A - α log x (MINUS sign)

✗ MISTAKE: Not arranging data in ascending order for Lorenz curve
  Must rank from POOREST to RICHEST

✗ MISTAKE: Using wrong Gini formula
  G = 1 - Σ(Xᵢ - Xᵢ₋₁)(Yᵢ + Yᵢ₋₁)
  NOT: G = 1 - Σ(Xᵢ - Xᵢ₋₁)(Yᵢ - Yᵢ₋₁)


NATIONAL INCOME
═══════════════

✗ MISTAKE: Confusing GDP and GNP
  GNP = GDP + NFIA (Net Factor Income from Abroad)

✗ MISTAKE: Wrong sign for NFIA
  If factor income received > paid: NFIA is POSITIVE
  If factor income received < paid: NFIA is NEGATIVE

✗ MISTAKE: Adding subsidies instead of subtracting indirect taxes
  GDP_FC = GDP_MP - Indirect Taxes + Subsidies

✗ MISTAKE: Forgetting depreciation for NNP
  NNP = GNP - Depreciation


INPUT-OUTPUT
════════════

✗ MISTAKE: Dividing by row total instead of column total
  aᵢⱼ = xᵢⱼ/Xⱼ (divide by COLUMN total)

✗ MISTAKE: Wrong matrix inversion formula
  For 2×2: Swap diagonal, negate off-diagonal, divide by determinant

✗ MISTAKE: Not checking Hawkins-Simon conditions
  Always verify |I - A| > 0 before inverting


ELASTICITY
══════════

✗ MISTAKE: Forgetting negative sign in price elasticity
  Price elasticity is typically NEGATIVE (inverse relationship)

✗ MISTAKE: Using wrong formula for arc elasticity
  Arc Ed = [(Q₂-Q₁)/(Q₂+Q₁)] / [(P₂-P₁)/(P₂+P₁)]
  NOT: [(Q₂-Q₁)/Q₁] / [(P₂-P₁)/P₁]

✗ MISTAKE: Wrong interpretation of elasticity values
  |Ed| > 1: Elastic (NOT Ed > 1, consider absolute value)
```
