# Introduction

The core problem in Machine learning is parameter estimation, this require solving an optimization problem, where we try to find the values for a set of variables $\theta $ $\in$ $\real^{D}$, where D is number of parameter

$$
\theta^{*} = \arg\min_{\theta \in \Theta} \mathcal{L}(\theta)
$$
